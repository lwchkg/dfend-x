Program options dialog page "QBasic"

On this page you can setup the path to your QBasic interpreter. If a QBasic interpreter is defined in D-Fend Reloaded you can choose bas files in the profile editor as program files. D-Fend Reloaded will automatically start the bas file inside QBasic inside DOSBox.
