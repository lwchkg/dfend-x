Menu:   Extras | Image files | Write IMG to floppy

Write IMG to floppy

This dialog will allow you to write IMG image files to real floppy disks.

IMG floppy images be read from real floppy disks via the "Create IMG image from floppy" menu item of the "Images" submenu of the "Extras" menu.

IMG floppy images can be mounted as drives inside DOSBox via the drives page of the profile editor.
