Create prof files

The prof-files are used by D-Fend Reloaded to store the profile information. The D-Fend Reloaded prof-file format is just an extension of the D-Fend prof-file format. For this most frontends that can import D-Fend profile file, will also be able to import D-Fend Reloaded prof-files.

You do not need to use the export funtion to get the prof-files. You can also just copy them from the Confs subfolder of the program data folder.

Export profile as auto setup templates

If this option is activated, the profile files will not only be copied to the destination folder but also the copies will be transformed to auto setup templates. Auto setup templates do not contain installation specific information (like the installation path) but all setup data needed to run the name and the meta information about the game.
