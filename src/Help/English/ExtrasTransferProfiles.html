Transfer profiles

From this dialog you can copy profiles from one installation to another (for example from your regular installation to a portable installation on a USB-stick). Transfering a profile means copying all files that belong to the profile (profile file, game program files, captured screenshots etc. and data files), so the game can be used from the destination installation without needing to have access the the source installation.

This feature can only be used on profiles that use relative path names, because when using absolute path names to locate game files, D-Fend Reloaded does not know how to copy them to the context of the destination installation.

Create portable D-Fend Reloaded installation in destination folder

If this option is turned off, you will need to have a D-Fend Reloaded installation in the selected destination folder; otherwise the transfer process will fail. If this option is turned on, D-Fend Reloaded will create an portable installation of itself in the destination folder before copying the profiles. Therefore it will copy its own program and configuration files (and will modify them for creating an portable installation).
