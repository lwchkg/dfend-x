Menu:   View | Show quick starter window

Quick starter window

From the quick starter window you can directly start any DOS program file inside DOSBox. The quick starter window looks like the Windows Explorer a bit: On the left side there is a tree structure and on the right side there is a file list.

Directory tree

In the directory tree on the left side of the quick starter window you can see a list of directories where your games or game related files can be stored. From the drop down box at the top of the tree structure you can choose which directories to show. In most cases it will be easier not to show the complete directory tree of your computer but only the games directory (defined on the directories page of the program options dialog) with it subdirectories.

By clicking on a directory on the left the file list on the right will show the content of the directory.

You can also open any directory inside DOSBox or with the Windows Explorer via the context menu of the directory tree.

File list

The file list on the right side of the quick starter window shows the content of the directory selected in the tree structure on the left.

You can open/start any file from the file list just by double clicking it. D-Fend Reloaded will decide automatically how the file should be started. (Text files will be opened in a text editor, Windows exeE files will just be started, DOS exe, com and bat files will be started inside DOSBox, images, sounds and videos will be played by the D-Fend Reloaded previewers.) For some files that can be started but also opened in a text editor (like html files or bat files) there is a context menu item to open the file in a text editor instead of running them the normal way.

When starting a file inside DOSBox via the quick starter the template and the settings from the bottom of the quick starter window will be used.

You can also directly make profiles from any DOS program file right via the context menu. There are two menu items "Make profile from file" and "Make profile from file (wizard)". The first item will open the profile editor with the chosen program file and the (on the bottom of the quick starter window) chosen template. The second will open the profile wizard with the chosen program file.

By dragging and dropping files or folders to the file list on the right of the quick starter window you can copy files and folders to the selected directory.

 

Hint:

The quick starter window is just a normal window and not a modal dialog. This means you can work on the D-Fend Reloaded main program window (filtering, changing, starting profiles etc.) while the quick starter window is opened.
