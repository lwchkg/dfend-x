Program options dialog page "Non English translations for the games list"

If you use D-Fend Reloaded with English user interface, the options offered on this page are not relevant for you.

This translations list allows you to display translated names in the games list for "genre", "language" and "language" even so the profile itself contains English names. This is very useful if you want to create profiles via auto setup templates because in the auto setup templates always English names for the genre and the game language are used.

If you enter translations for you language for genres, game languages and licenses, D-Fend Reloaded will translate the names in the games list on the fly. The names will also be translated in the profile editor but they will still be stored in English in the profile itself.
