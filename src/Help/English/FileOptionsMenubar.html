Program options dialog page "Menubar"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can change turn the menubar on the D-Fend Reloaded main window on and off. To restore the the menubar, use the context menu of the toolbar. For always being able to restore the menubar, you cannot hide the menubar and the toolbar at the same time.
