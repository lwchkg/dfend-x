Program options dialog page "Columns in the games list"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can select which columns should be visible on the games list in view mode report and in which order (from left to right) they should appear.

In the default setup all columns except the "Notes" and "Number of starts" column will be shown in the order "Setup", "Genre", "Developer", "Publisher", "Year" and "Language".

  • Show grid lines in view style "report"
    If this option is activated, and the view style "report" is chosen, D-Fend Reloaded will show grid lines between the coloumns and rows in the games list.
    (Default value for this option: off)
  • Restore column widths on program start
    If this option is activated, the column widths in the games list which have been modified by the user will be restored on the next program start.
