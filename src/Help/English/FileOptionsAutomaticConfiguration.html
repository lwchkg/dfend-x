Program options dialog page "Automatic game configuration"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can set up when D-Fend Reloaded should configurate a game completely automatically.

  • Archive file import: Do not show dialog if the game can be identified by an auto setup template
    If this option is checked D-Fend Reloaded will not show any dialog if a game which is going to be imported via File|Import|Archive or via drag&drop can be identified by an auto setup template. If there is no matching auto setup template the dialog to select a template will be shown anyway.
    (Default value for this option: on)
  • Wizard mode
    In this box you can select how many questions the profile wizard should ask when added a new game. It can show the pages for selected a profile and for entering game meta information always, only if there is no matching auto setup template or never.
    (Default value for this option: "Only setup game automatically if matching auto setup template exists")
  • Program file names to identify games that need to be installed
    If there is only one program file matching to one of the records in this list D-Fend Reloaded will suggest to import the game using the game installation support.
