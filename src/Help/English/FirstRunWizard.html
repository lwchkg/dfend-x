First run wizard

The first run wizard will help you to initially setup D-Fend Reloaded.

Program language and DOSBox language

The program and the DOSBox language will initially be chosen from the setup program so in most cases this two fields will already show the right language.

DOSBox installation to be used by default

In diesem Feld können Sie angeben, welche DOSBox-Installation standardmäßig zum Aufruf der DOS-Spiele verwendet werden soll. Sofern Sie bei der Installation von D-Fend Reloaded die Option, DOSBox mit zu installieren, nicht explizit abgewählt haben, können Sie die Einstellung "Mit D-Fend Reloaded installierte Version" beibehalten. Unabhängig von der hier gewählten Standard-Installation können Sie später jedem Profil auch eine individuelle DOSBox-Version zuweisen.

Automatically check for updates

Via this checkbox you can choose whether D-Fend Reloaded should look for updates automatically. (The update checker will only search for updates and inform you about new versions, it will never automatically install any update without asking you.) If you keep to default setting D-Fend Reloaded will never connect to the internet automatically. You can look for updates manually at any time via the Search for updates function from the help menu.

You can change all settings later at any time via the program options dialog.
