Import archive file

With this function you can import archive file based complete profiles (including all game files), plain archive files and archive files made with DBGL into D-Fend Reloaded. You can create archive files via the Make archive package function of the "Profile" menu.

Import via drag&drop

You can also import archive files by just dragging them from the Explorer to the D-Fend Reloaded program window.
