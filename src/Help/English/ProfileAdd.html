Menu:   Profile | Add

Add DOSBox profile

Clicking this option will create a new DOSBox based profile and open it in the profile editor dialog. (If you have chosen to use the classic profile editor dialog this one will open, otherwise the modern profile editor dialog will be used.)

If you want to add a ScummVM based profile or a link to a Windows game, you have to use the menu items Add ScummVM profile and Add Windows game profile from the "Profile" menu.
