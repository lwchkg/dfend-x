Profile editor dialog page "DOSBox settings"

On this page you can configurate some DOSBox specific settings.

  • DOSBox priority (foreground)
    This is the priority Windows should give DOSBox when it is running in foreground. Normal Windows applications will get priority "normal". Giving DOSBox a higher priority will give more CPU power to DOSBox than to other programs.
  • DOSBox priority (background)
    This is the priority Windows should give DOSBox when it is running in background. Normal Windows applications will get priority "normal". Giving DOSBox a higher priority will give more CPU power to DOSBox than to other programs.
  • Close DOSBox when program closes
    If this option is activated, the DOSBox window will be closed when the program running inside DOSBox quits. Otherwise the DOSBox window will stay open and you will have to enter "exit" to close it manually. If the program you want to run inside DOSBox terminates with an error turning off this options allows you to read the error message before DOSBox closes.
  • Use predefined DOSBox installation
    In this box you can select the DOSBox installation you want to use. The DOSBox installation listed in this box can be configurated on the DOSBox page of the program options dialog.
  • Use this custom DOSBox installation
    In this edit field you can define a custom directory for a DOSBox version to use.
  • Use this custom settings for the console window
    By activating this option you can overwrite set setting to show or to hide the DOSBox console made on the DOSBox page of the program options and define if the console shall be shown always or never for the current profile.
  • Use this custom language in DOSBox By activating this option you can choose a custom language for DOSBox messages when running this profile. If no custom language is activated the language selected on the DOSBox page of the program options dialog will be used.
  • Custom settings
    The lines entered in this edit field are added to the DOSBox configuration file when running DOSBox. You can overwrite all other settings by writing something in this box. Also settings perhaps offered by some newer DOSBox versions can be used by writing the configuration lines to this field.
