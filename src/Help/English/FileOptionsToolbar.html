Program options dialog page "Toolbar"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can change the appearance of the toolbar in the main window.

  • Show toolbar
    Will turn the toolbar in the main window completely on or off. The the toolbar ist turned of all other options on the page will have no effekt.
    (Default value for this option: on)
  • Visible buttons
    In this check-listbox you can select the buttons to be visible on the toolbar in the main window.
    (Default value for this option: all but "Close" and "Run setup")
  • Show toolbar captions
    If this option is activated icons and names are shown on the toolbar buttons. Otherwise only the icons will be visible.
    (Default value for this option: on)
  • Function of the add button
    In this dropdown box you can select what should happen if you click on the "Add" toolbar button. You can choose between:

    • "Add dialog": Will open the regular profile editor for adding a new profile.
    • "Profile wizard": Will open the profile wizard for adding a new profile.
    • "Selection menu": Will show a dropdown menu from which you can select what to do.

    (Default value for this option: "Selection menu")
  • Background image
    In this edit field you can enter the name of an image file that should be displayed as a background image on the toolbar in the main window.
    (Default value for this option: no background image)
  • Font size
    From this field you can choose the font size the toolbar captions shall be displayed in.
    (Default value for this option: 9)
