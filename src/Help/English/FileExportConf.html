Create conf files

The conf-files are used by DOSBox as configuration files. Most other frontends can import conf-files. So if you want to transfer profiles to an other frontend using conf-files for this will probably work. But conf-files contain less information than prof-files. So if the other frontend can read D-Fend / D-Fend Reloaded prof-files you will get besser results when importing them instead of the conf-files.

If you just want to look inside a conf-file used by D-Fend Reloaded when calling DOSBox, you do not need to use the export function. You can open the conf-file for the selected profile via the menu function View DOSBox conf file from the "Profile" menu.
