Profile editor dialog page "Starting"

On this page you can setup what DOSBox should do when it starts (that means you can setup here the way DOSBox will start the chosen game).

  • Override game start
    If this option is activated the program file and the setup file configurated on the Profile page are ignored when starting the profile.
  • Override mounting
    If this option is activated the settings made on the Drives page are ignored when starting DOSBox. This means no drives will be mounted when starting DOSBox.
  • Autoexec.bat
    The commands entered in this box will be executed when starting DOSBox before starting the game. The lines entered in this box have the same meaning as the Autoexec.bat in real DOS.
  • Finalization
    The commands entered in this box will be executed after executing of the game before closing DOSBox.
  • Start DOSBox normally
    If this option is activated the game or setup file entered on the Profile page will be started when starting the profile. This should be the right setting for all most all games. You should only use one of the two following options if you want to boot an image file.
  • Start hard disk image
    If you have configurated a hard disk image without a DOSBox drive letter on the Drives page, you can setup to boot from this image from the dropdown box next to this option.
  • Start floppy image
    By activating this option you can boot from any floppy disk image. There is no need to mount the floppy images you want to boot from. You can also specifiy multiple floppy images for booting. DOSBox will use the first image for inital boot and you can swap the virtual floppies in the virtual drive by pressing Ctrl+F4.
