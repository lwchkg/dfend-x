Profile editor dialog page "Keyboard"

On this page you can setup the virtual keyboard DOSBox should emulate and the keyboard driver, DOSBox should provide to the programs running inside DOSBox.

  • Use scan codes
    By activating or deactivating you can change the method DOSBox will use to get the information about the pressed keys on the real keyboard. Activating or deactivating this option can cause or solve some keyboard problems depending on your keyboard and your operating system.
  • Keyboard layout
    In this dropdown box you can select the keyboard layout you want to use in DOSBox. You should use the layout of your real keybaord. If you select "default" the keyboard layout defined for the chosen DOSBox installation (or if there is also "default" selected) in the current language file will be used. (The keyboard layout chosen in this case will be shown in the information text below the dropdown box.) Selecting "default" will make DOSBox use the "US" layout when the D-Fend Reloaded program language is English, layout "GR" (=German) when the D-Fend Reloaded program is German etc.
  • Codepage
    In this dropdown box you can select the codepage you want to use in DOSBox. The codepage influences how characters are displayed on the DOSBox command line. If you select "default" the codepage defined for the chosen DOSBox installation (or if there is also "default" selected) in the current language file will be used. (The codepage chosen in this case will be shown in the information text below the dropdown box.)
    Attention: Not all codepages are supported for all keyboard layouts. You can find a list of all possitible keyboard layout and codepage combinations here.
  • Num lock status, Caps lock status and Scroll lock status
    Via this three boxes you can select the initial state of the three toggle keys num lock, caps lock and scroll lock. The option "Do not change" means DOSBox will use the state given by the Windows keyboard device driver as the initial state of the three virtual toggle keys.
  • Use default DOSBox key mapping file
    DOSBox uses a key mapping file for storing which physical key shall be connected with which virtual key inside DOSBox. If this option is activated the key mapping file defined on the DOSBox page of the program options dialog will be used.
  • Use this custom DOSBox key mapping file
    If this option is activated the given key mapping file will be used inside DOSBox instead of the regular key mapping file.

Hint:

You can open the key mapper to change the mapping from physical to virtual keys inside DOSBox at any time by pressing Ctrl+F1.
