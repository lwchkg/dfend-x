Profile editor dialog page "Joystick"

On this page you can setup the virtual joystick DOSBox should emulate. DOSBox will map this virtual Joystick to any real Joystick connected to your computer and supported by Windows automatically.

  • Joystick type
    From this drowpdown box you can select the type of joystick DOSBox should emulate:

    • auto:chooses emulation depending on real joystick(s)
    • none: disables joystick emulation
    • 2axis: supports two joysticks
    • 4axis: supports one joystick, first joystick used
    • 4axis_2: supports one joystick, second joystick used
    • fcs: Thrustmaster
    • ch: CH Flightstick
  • Timed intervals for axis
    Enable timed intervals for axis.
  • Autofire
    If this option is activated, DOSBox will send separate fire-button press events to the game as long as you keep the button pressed.
  • Swap axis 3 and 4
    If this option is activated the 3rd and 4th axis phyisical axis of the connected joystick will be swaped in the emulation.
  • Activate button wrap
    If this option is activated, on joysticks with more physical buttons than emulated virtual buttons (see "type" dropdown box) the higher physical buttons are mapped to lower logical buttons in the emulation. For example if you use a joystick with more than 4 buttons and you have configurated in the "type" dropdown box a 4-button joystick, physical button 5 is mapped to logical button 1, physical button 6 to logical button 2 etc.
