Profile editor dialog page "Serial ports"

On this page you can setup the virtual serial ports DOSBox should offer to the game running inside it. There are 5 different operation modes for each emulated serial port:

  • Disabled
    Support for the selected serial port will be disabled.
  • Dummy
    Will create a dummy serial port inside DOSBox without any real function.
  • Modem
    Will emulate a virtual modem connected to the serial port inside DOSBox. The real connection will be established via the network of the real computer. Therefore you will have to choose "Listen for connections" on the server.
  • Nullmodem
    Activating this option will emulate a virtual nullmodem cable connected to the serial port inside DOSBox. The "other side" of the cable must be connected to another DOSBox instance running on another computer. One computer will be the server and one the client. On the client computer you will have to specifiy the address of the server computer.
  • Direct serial
    This option will connect the selected virtual serial port inside DOSBox with a real real serial port of the real computer.
