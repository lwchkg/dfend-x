Program options dialog page "Profile editor"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can set up which profile editor style you want to use and some tweaks for the profile editor.

  • Reopen last active profile editor page
    If this option is checked the profile editor will open on the same page which was active when it was closed the last time.
    (Default value for this option: off)
  • Use classic D-Fend style profile editor / Use modern profile editor
    Via this two radio buttons you can choose which profile editor D-Fend Reloaded should use. The classic D-Fend style profile editor mostly looks like the profile editor from D-Fend. You will find in the classic profile editor the same settings on the same pages as in the D-Fend profile editor. The classic profile editor is only ment for compability to D-Fend and does not cover all new settings available in D-Fend Reloaded. In most cases using the modern profile editor should be the right choise.
    (Default value for this option: "Use modern profile editor")
  • Setup screenshot folder automatically when adding a profile
    In this box you can choose if you want the modern profile editor to create screenshot folders for a new profile automatically.
    (Default value for this option: "When adding via the profile wizard and via the modern profile editor")
  • Rename profile file when renaming profile automatically
    If this option is off, the profile filename will be derived from the initially chosen profile name and not changed if you rename the profile. (But you can change the profile file name at any time from the profile editor.) If this option is on, you can not change the profile filename manuelly anymore but D-Fend Reloaded will change the profile filename each time you change the profile name.
    (Default value for this option: off)
  • Automatically add mounting records if needed
    In this box you can choose if you want the modern profile editor to create the required mounting settings automatically if the selected game file cannot be reached by the mountings.
    (Default value for this option: "When adding via the profile wizard and via the modern profile editor")
