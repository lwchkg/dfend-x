Program options dialog page "List in screenshot mode"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can set up the appearance of the games list if view mode screenshots is selected.

  • Width and height of the screenshots
    Via this two field you can define the size to which the screenshots shall be scaled when displaying in the games list.
    (Default value for this option: width 150 pixels, height 100 pixels)
  • Use screenshot with the following number if no screenshot is selected
    If this option is activated the screenshot with the number selected from the edit field below of the game is used for displaying in the games list if no special screenshot is selected for the screenshots games list. If this option is turned off and no default screenshot is defined for a game the game icon or if no game icon is defined the default icon is shown in the games list instead of a screenshot.
    (Default value for this option: on / number 1)
