Old games

From this submenu you can open old games related internet sites.

Edit links

By clicking the menu item you can open the edit links dialog. In this dialog you can configurate the names or the links and corresponding the internet pages. Each line represents one internet link; on the left side you can enter the name to display and on the right side the internet address to open. Empty lines will result in dividing lines. By starting a name with a "=" sign you can create submenu items.
