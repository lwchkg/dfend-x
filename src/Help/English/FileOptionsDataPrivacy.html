Program options dialog page "Data privacy"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can setup whether D-Fend Reloaded should record history information on the profiles you have started. You show the recorded information and optionally delete them.

The recorded information are stored locally only and can be viewed via the "View history" dialog.
