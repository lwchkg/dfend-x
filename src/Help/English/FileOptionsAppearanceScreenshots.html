Program options dialog page "Media preview"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can change the appearance of the screenshots etc. preview area at the bottom of the main window.

  • Background color / image
    Via the three radio buttons "Default background color", "Background color" and "Background image" you can set up whether D-Fend Reloaded should you the default Windows background color in the screenshot, sounds, videos and data preview area, a custom color or a custom image.
    (Default value for this option: "Default background color")
  • Font size and font color
    From this two fields you can choose the font size and color for the texts in the preview area.
    (Default value for this option: font size 9, font color "Window text")
  • Screenshot preview size
    The screenshots in the preview area will be scaled (keeping the proportions) to fit in an area of the width and height defined in this field.
    (Default value for this option: 100)
