Profile editor dialog page "Hardware"

On this page you can configurate which hardware should be used by ScummVM. For DOSBox profiles this page will be empty.

  • CD drive for playing audio (for games using CD audio)
    In this box you can define which CD drive should be used for playing music from in the game. The number of the first CD drive is 0.
  • Use Joystick number
    If the game supports joysticks for input you can choose from this edit field which joystick ScummVM should use for user input. The number of the first joystick conntected to the system is 0.
  • Platform
    In most cases the default value "auto" should not be changed. In "auto" mode ScummVM will try to auto detect the platform of the game files. Only if the game is not running well you should select the platform for which the game was made here.
