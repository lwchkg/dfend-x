Menu:   Run | Run DOSBox

Run DOSBox

This menu item will run DOSBox with it's default configuration file from the DOSBox directory. You can open the DOSBox default configuration file in an editor via the Open DOSBox configuration file menu item from the "Run" menu.

The default DOSBox configuration file from the DOSBox directory has no meaning for D-Fend Reloaded. It will no be used when creating profile. You can change the configuration file template for new profiles via the Templates menu item from the "Extras" menu.
