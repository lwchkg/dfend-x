View modes

Via the seven following items of the view menu you can set up the appearance of the games list:

  • List (no icons):
    Will show only the names of the profiles. The profiles are ordered in columns.
  • List:
    Will show only the names and the icons of the profiles. The profiles are ordered in columns.
  • Report (no icons):
    Will show many information about each profile (Genre, Developer, etc.). You can set up on the "Columns in the games list" page in the program options dialog which columns should be shown. The profiles are ordered in one column.
  • Report:
    Will show many information about each profile (Genre, Developer, etc.) including the profile icon. You can set up on the "Columns in the games list" page in the program options dialog which columns should be shown. The profiles are ordered in one column.
  • Small icons:
    Will show only the names and the icons of the profiles. The profiles are ordered in rows.
  • Icons:
    Will show larger icons for each profile. The icons are ordered in rows.
  • Screenshots:
    Will use screenshots from the profile capture folders instead of the icons to show in the games list. The screenshots are ordered in rows. (See also page "List in screenshot mode" in the program options dialog.)

 

Hint:

You can choose the screenshot to show in screenshot view mode via the context menu of the screenshot previewer at the bottom of the program window.
