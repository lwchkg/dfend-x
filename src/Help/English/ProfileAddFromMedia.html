Install from source media

By using this dialog you can add games to the D-Fend Reloaded games list that need to be installed before being useable. In the dialog you can choose on which type of medium the game files are available and select the source media files. After Clicking "Ok" D-Fend Reloaded will mount the installation media temporarely and run DOSBox in window mode. If D-Fend Reloaded finds an unique installation program on the first installation medium, it will automatically run this installer. All you have to do is follow the commands of the installer and install the game in any folder on C:. When the installation is done D-Fend Reloaded will automatically search in the games folder for a new game and add it to the games list.
