Menu:   Profile | Copy

Copy profile

Instead of creating new profiles for each game, you can also copy existing profiles. In the "Copy profile" dialog there are two optional settings:

  • Set new screenshot folder
    If this checkbox is marked, the new game will get a new screenshot folder name. Otherwise the screenshot folder name will also be copied from the source profile and both profile will share the same screenshots folder.
  • Remove data folder record
    If this option is activated, the link to the data folder will be removed from the new profile. This means the source and the destination profile will not share the same data folder.
