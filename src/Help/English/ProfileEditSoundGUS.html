Profile editor dialog page "GUS"

On this page you can setup the virtual Gravis Ultra Sound (GUS) card. The settings on this page should match to the settings you have selected in the game setup program for the sound output.

To use the Gravis Ultra Sound support inside DOSBox you will need some external patch files. The path to this patch files can be entered in the edit field named "path". In this directory there should be a MIDI directory that contains the patch files for GUS playback.
