Adding games to D-Fend Reloaded

First of all it is recommended to read the main introduction to D-Fend Reloaded. It will explain to you how DOSBox and D-Fend Reloaded work.

The text on this page will describe how to add games to D-Fend Reloaded. There are mainly two ways for adding new games: Completely automatically and wizardbased half manual. To add a new game, which is stored as a zip file, completely automatically, just drag the game zip file to the D-Fend Reloaded program window. The zip file will be extracted to a new created subfolder of the games directory and the profile will be created automatically (based on an auto setup template if there is a auto setup template matching to the game). If you want to influence the palce where the game files are stored and want to influence the profile creation process, you should add the game by the following two steps:

1. Copy the games files to the games folder

  • Click on the "Open games folder..." from the "Extras" menu. An Explorer window will open and show you the default location for storing games. (Of course you can use any location you want, but as long as you do not know exactly how D-Fend Reloaded works, you should just use this folder.)
  • Create a new subfolder for the game in this directory.
  • Copy the game files to this new subfolder.

2. Add a profile for the new game

  • Click on the "Add with wizard..." menu item from the "Profile" menu.
  • The new profile wizard will open and ask you for the type of the new profile. If you want to run some old DOS game choose "DOSBox".
  • On the next page the wizard will ask you for the game file to start. Just select the game program file from the new folder you have created in step 1. (Clicking on the "Select file" button on the right side of the game file edit field will bring you directly to the default games location.)
  • As the last step you will have to choose a template for the new game (see also help page for the new profile wizard).
  • On the last two pages of the profile editor normally no changes are needed.
