Menu:   Profile | Edit | Directories

Profile editor dialog page "Directories"

On this page you can configurate additional files for the profile.

  • Screenshot folder
    The folder defined in this edit field will be used by DOSBox / ScummVM for storing screenshots, captured sounds and videos. You can also add files manually to this folder via the screenshot/sound/video viewer at the bottom of the main program window.
  • Data folder
    The optional data folder can be used to store manuals etc. for the game. The data folder can be opend in the Explorer at any time via the context menu of the profile and the content of the data folder can be accessed directly via the data folder viewer at the bottom of the main program window.
  • Additional program files and directories
    In this two boxes you can define files and folders belonging to the program but located outside the program folder. This will allow D-Fend Reloaded to transfer and uninstall this files together with the main program files.
