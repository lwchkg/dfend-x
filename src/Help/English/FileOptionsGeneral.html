Program options dialog page "General"

There are two checkboxes on this page:

  • Minimize to tray area
    If this option is activated, the D-Fend Reloaded program window will be visible in minimized state as an icon in the tray area (the area next to the clock on the right side of the taskbar) instead of a normal taskbar record.
    (Default value for this option: off)
  • Start with Windows (*)
    If this checkbox is checked, D-Fend Reloaded will be started together with Windows.
    (Default value for this option: off)
  • Allow only one running instance (*)
    Allows only one instance of D-Fend Reloaded running at the same time. If D-Fend Reloaded is started while an instance is already running, this running instance is just sent to foreground.
    (Default value for this option: on)

The options marked with (*) are only visible in advanced mode.
