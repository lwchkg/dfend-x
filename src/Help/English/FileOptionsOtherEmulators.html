Program options dialog page "Other emulators"

On this page you can setup other emulators (besides DOSBox and ScummVM). The emulators have to be Windows prgrams (this means the emulators have to be runable directly by Windows, not like QBasic which needs to be run through DOSBox). D-Fend Reloaded only supports the platforms DOSBox and ScummVM completely. The emulators defined on this page are only supported as Windows programs. This means if you add a game for one of the platforms defined on this page, D-Fend Reloaded will create a Windows game profile for running the emulator and will give the selected game data file as a parameter to the emulator.

Parameter

In this field you can enter the parameters that should be used when running the emulator. %s stands for the file name of the data file to be opened inside the emulator. Attend that many programs need the %s to be quoted to be able to handle path and filenames containing spaces while some other programs do not like this quotes. Please see the help file of the emulator you want to use for the right parameter usage.
