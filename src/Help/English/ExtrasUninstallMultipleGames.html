Uninstall multiple games

From this dialog you can uninstall multiple games at once. This means not only the profile files will be removed but also all game files can be removed. If you just want to delete the D-Fend Reloaded profiles while keeping the game files, you have to choose the the menu item "Delete" from the "Profile" menu.

Actions

In this box you can choose which files should be removed. If you want to delete a game you can keep the default option "Delete profile record and all program folders" activated. D-Fend Reloaded will only delete files that are not used by any other games and that are located in the base folder or its subfolders (see directories page of the program options dialog). This will ensure that even uninstalling of misconfigurated games cannot damage you software installation outside D-Fend Reloaded. (You can turn off this protection on the security page of the program options dialog if you really want to.)
