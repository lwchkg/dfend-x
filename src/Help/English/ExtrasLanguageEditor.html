D-Fend Reloaded language editor

The language editor helps you translating D-Fend Reloaded to another language. (You can also copy the language ini file and edit this with any text editor, but the language will be more comfortable.)

From the dropdown box at the top of the editor dialog you can choose the category to translate. The table will show three columns: In the first column you will find the identifiyer of the string to translate. This will be names like "File.Export" for the word "Export" in the "File" menu. In the second column you will find the english default translation and in the third, editable column you can enter the for the string in your language.
