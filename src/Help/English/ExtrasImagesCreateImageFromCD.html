Create ISO image from CD

This dialog will allow you to create ISO image files from real CDs. ISO images can be mounted as drives inside DOSBox via the drives page of the profile editor.

The D-Fend Reloaded ISO image reader is only capable to read the first track of a CD. If you want to make an ISO image of a CD with multiple tracks (one data track followed by some sound tracks), you should use some exteral ISO image reader for creating the ISO image.
