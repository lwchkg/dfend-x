Import prof file

prof-files are used by D-Fend Reloaded and also by D-Fend (classic) for storing the profile information. This program option simply copies the chosen prof-file to the D-Fend Reloaded Confs folder. You can also import prof-files by just copying them to this folder.

Importing profiles from D-Fend

If you want to import profiles from an D-Fend installation, you can import the conf-files or the prof-files. Because the prof-files contain more information it is recommended to import the prof-files.

Also D-Fend and D-Fend Reloaded are using the same file format, D-Fend Reloaded will store additionally information. When using a D-Fend Reloaded profile in D-Fend (classic) this should ignore the unknown keys.

Import via drag&drop

You can also import conf and prof files by just droping them from the Explorer to the D-Fend Reloaded program window.
