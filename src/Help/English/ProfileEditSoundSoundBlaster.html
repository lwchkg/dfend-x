Profile editor dialog page "SoundBlaster"

On this page you can setup the virtual SoundBlaster card. The settings on this page should match to the settings you have selected in the game setup program for the sound output.

Meaning of the "Type" dropdownbox

Type Name Bits Stereo
none SoundBlaster emulation disabled    
sb1 SoundBlaster ver. 1.0 8 no
sb2 SoundBlaster ver. 2.0 8 no
sbpro1 SoundBlaster Pro ver. 1.0 8 yes
sbpro2 SoundBlaster Pro ver. 2.0 8 yes
sb16 SoundBlaster 16 16 yes
