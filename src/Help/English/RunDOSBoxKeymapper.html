Menu:   Run | Run DOSBox keymapper

Run DOSBox keymapper

Clicking this menu item will start DOSBox and will start the keymapper inside DOSBox immediately.

The keymapper

The keymapper will allow you to map physical keys to virtual keys inside DOSBox. This can fix many keyboard problems. But you do not need to use the keymapper to setup a keyboard layout (for a language) inside DOSBox. You can do this much easier via the profile editor dialog by just selecting the desired language.

The keymapper file

The mapping settings will be stored in a key mapping file. You can setup the path and the name of this file via the program options dialog. This mapper file will be used by all profiles by default but you can also setup any other custom mapper file for any profile.
