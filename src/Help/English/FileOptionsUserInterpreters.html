Program options dialog page "User-defined interpreters"

On this page you can setup uses-defined interpreters for some file types. The interpreter programs have to be DOS programs, this means they have to be executed inside DOSBox. For each interpreter you can specify the parameters needed to load and run the data files and the file types for which it applys. If you specifiy an extension here, you will be able to select files of this types in the profile editor as program files. D-Fend Reloaded will execute this files automatically using this interpreter in DOSBox.
