Menu:   File | Program options | Service | Reset profiles to templates

Reset multiple profiles to values from templates

Using this dialog you can reset some settings for one or more profiles to the default value from a profile. The dialog has three pages: On the first page you can select the profile which you want to reset, on the second page you can select the settings to be resetted and on the third page you can choose the template to which you want to reset the settings of the profils.
