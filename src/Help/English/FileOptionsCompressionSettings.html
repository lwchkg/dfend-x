Program options dialog page "Compression settings"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can choose the compression strength D-Fend Reloaded should use when repacking folders (used as drives in DOSBox) to zip files. A more strength compression will result in a smaller archive file but will take a longer time.

On most systems the default value "Good compression" should be kept. The maximum compression level will not bring much smaller archive files but compression will take much longer.
