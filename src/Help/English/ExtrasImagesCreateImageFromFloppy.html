Menu:   Extras | Image files | Create IMG image from floppy

Create IMG image from floppy

This dialog will allow you to create IMG image files from real floppy disks. IMG floppy images can be mounted as drives inside DOSBox via the drives page of the profile editor.

IMG floppy images can also be written to floppy disks again via the "Write IMG image to floppy" menu item of the "Images" submenu of the "Extras" menu.
