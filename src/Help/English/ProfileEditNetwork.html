Menu:   Profile | Edit | Network

Profile editor dialog page "Network"

On this page you can configurate to connect to DOSBox instances on different computers via a network. DOSBox will offer a virtual UDP/IP network adapter inside it for the game.

  • Activate UDP/IP emulation
    Will turn the whole network emulation inside DOSBox on or off. If this option is deactivated, all other options on this page will have no effect.
  • Establish IPX connection
    From this box you can select if the current computer should be the server or the client.
  • IPX server IP address (for Client-mode)
    If you have chosen the current computer to be the client you can enter the IP address of the server to connect to in this edit field.
  • IPX Port
    From this box you can select the port DOSBox should use for connection. The port on the client and on the server must be the same for a connection.
