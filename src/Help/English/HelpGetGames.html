How to get games

There are may ways to legally get old DOS games:

Old floppy disks

If you still own old DOS games on floppy disks and have a computer with a floppy disk drive, you can directly use this floppy disks:

  • You can copy the contents of the floppy disks to hard disk folders and then import the games via the menu item "Import from source media" in the "Profile" menu. (Choose "Install from folder" in the dialog.)
  • If you ware running D-Fend Reloaded on the computer with the floppy disk drive, you can directly install the games from the flopy disks. Choose "Import from source media" in the "Profile" menu and then select "Install from real floppy disks".

D-Fend Reloaded package manager

The menu item "Download packages" from the "Import" submenu of the "File" menu will open the D-Fend Reloaded package manager. The package manager offers you many freeware and shareware games for direct download. All you have to do is choose the games you want to have and click on "Download and install selected games". There is no need to configurate anything on games installed this way. These games are at once ready to use.

Freeware and shareware on the internet

On the "Old games" submenu of the "Help" menu you will find a list of many internet pages offering DOS games and programs (freeware and shareware) for download. In many cases the games can be dragged and dropped directly to the D-Fend Reloaded program window and being downloaded and installed by just doing this. (But you can also download the archive files and then drag&drop the archive files to the program window or use the menu item "Import archive file" from the "Import" submenu of the "File" menu.) If D-Fend Reloaded already has a matching auto setup template for the game, the game will be configured automatically and will be ready at once. In the other case you will be asked for a name for the game and which template to use.

Buying games

The internet site www.gog.com offers many old DOS games for cheap prices.
