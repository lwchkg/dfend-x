Profile editor dialog page "Sound"

On this page you can configurate the main sound card emulation settings for DOSBox. inside DOSBox.

  • Activate sound
    If this option is deactivated the whole sound support inside DOSBox will be disabled and all other options will have no effect.
  • Mixer
    In this box you can setup the DOSBox internal mixer. DOSBox will collect the sound output of the game in its mixer and send it in the given format to the real sound card. In most cases the default values should not be changed.
  • Activate PC speaker and PC speaker sample rate
    If this option is activated DOSBox will emulate a virtual PC speaker for the games. The sounds a game want to play though the virtual speaker device will be redirected to the real sound card.
  • Tandy speaker and Tandy speaker sample rate
    By changing this settings you can influence the Tandy speaker emulation. In most cases the default values should work very well.
  • Activate Disney Sounds Source
    By checking this option you can activate the Disney Sounds Source support.
