DOSBox language editor

The language editor helps you translating DOSBox to another language. (You can also copy the language lng file and edit this with any text editor, but the language will be more comfortable.)

From the dropdown box at the top of the editor dialog you can choose the string to translate. The text box will show the value of the selected key.
