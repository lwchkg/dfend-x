Import folder

With this function you can import whole folders to the D-Fend Reloaded file structure. D-Fend Reloaded is copying the folder to the game files folder and creating a new game record for the program in the folder.

Import via drag&drop

You can also import folders by just dragging them from the Explorer to the D-Fend Reloaded program window.
