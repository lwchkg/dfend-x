Menu:   Profile | Uninstall

Uninstall profile

Clicking this menu item will uninstall the selected profile from D-Fend Reloaded. You can choose in the dialog which components you want to be removed.

From the uninstall dialog you can choose which game files to delete. If you want to delete a game you can keep all options listed in this dialog activated. D-Fend Reloaded will only offer files to delete that are not used by any other games and that are located in the base folder or its subfolders (see directories page of the program options dialog). This will ensure that even uninstalling of misconfigurated games cannot damage you software installation outside D-Fend Reloaded. (You can turn off this protection on the security page of the program options dialog if you really want to.)
