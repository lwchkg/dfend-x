Program options dialog page "Directories"

There are there input lines on this page:

  • Base folder (for relative path)
    All paths in D-Fend Reloaded will be stored relatively to this path. This folder should be the base for all D-Fend Reloaded data files. The program will only be able to perform operations like transfering profiles or building installer packages if the game files are in this directory structure.
    (The default value for this option depends on the chosen operation mode. In operation mode "Program directroy mode" and in "Portable mode" the default value for this folder is the program folder. In operation mode "User directroy mode" the default value for this folder is C:\Users\YourAccount\D-Fend Reloaded.)
  • Game folder
    When clicking in the profile editor on the "Choose file button" for the game exe file or the setup exe file, this folder will be offered to choose a file first.
    (Default value: Subfolder "VirtualHD" of the base folder.)
  • Additional data folder
    When adding an additional data folder via the "Create default game data folder" button in the profile editor a subfolder in this folder is created.
    (Default value: Subfolder "GameData" of the base folder.)
  • Default base folder for capture directories
    D-Fend Reloaded will create the folders for the screenshots made by DOSBox as subfolders of this directory.
    (Default value: Subfolder "Capture" of the base folder.)

There is no need to use one of this folders when storing game files. You can setup to run any file from any directory in the profile editor, but some operations like transfering profiles or building installer packages can only be performed on games with all files in subfolders of the base folder.

The "Game folder" and the "Additional data folder" are only used as initial folders in some file/folder select dialogs.

Directories in protable mode

In portable mode this three directories are stored relatively to the program folder at program end. When D-Fend Reloaded is started again it will make absolute paths for the three directories again. So the directories on this program options page are shown with absolute paths in any operation mode.
