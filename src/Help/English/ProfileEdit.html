Edit profile

The profile editor allows you to modify all settings belonging to a profile. Because there are many settings to configurate for each profile the profile editor dialog has been divided into pages. On the left side of the dialog there is a tree structure. Clicking on any of the items on the left will change the page shown on the right side of the dialog.

Pages of the profile editor dialog

Because the profile editor is used to configurate DOSBox, ScummVM and Windows game profiles, not all pages are shown at the same time. (The pages shown in DOSBox mode are marked with "D", the pages shown in ScummVM mode are marked with "S", pages shown in Windows game mode are marked with "W" and the pages shown when editing templates only are marked with "T".)

  • Profile (D, S, W)
    • Additional checksums (T)
    • Program information (D, S, W)
    • Directories (D, S, W)
    • Helper programs (D, S, W)
  • DOSBox settings (D)
  • ScummVM settings (S)
  • Hardware (D, S)
    • CPU (D)
    • Memory (D)
    • Graphics (D, S)
    • Keyboard (D)
    • Mouse (D)
    • Sound (D, S)
      • Volume (D)
      • SoundBlaster (D)
      • GUS (D)
      • MIDI (D)
      • Innovation SSI-2001 (D)
    • Joystick (D)
    • Drives (D)
    • Serial ports (D)
    • Network (D)
    • Printer (D)
  • DOS environment (D)
  • Starting (D)

To add ScummVM based profiles you will first need to install ScummVM and configurate it on the ScummVM page of the program options dialog.

Classic D-Fend v2 style profile editor

There is also a classic profile editor for compability with D-Fend v2. This classic profile editor does not cover all settings available in D-Fend Reloaded. Additionally it can not be used on ScummVM and Windows game profiles. If you do want to have the D-Fend v2 look and feel, you should use the modern style profile editor. You can change to profile editor to use via the profile editor page of the program options dialog.
