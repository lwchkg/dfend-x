Edit DOSBox profile (Classic profile editor)

The classic profile editor is only available for compatibility with D-Fend v2. You should only use this profile editor if you want your D-Fend Reloaded look and feel like D-Fend v2. The classic profile editor does not cover all the new D-Fend Reloaded features.

You can define one the profile editor page of the program options dialog whether to use the classic or the modern profile editor.

For a description of the settings offered in the profile editor please the help page for the modern profile editor.
