Program options

The program options dialogs allows you to customize a lot of things inside D-Fend Reloaded. To manage the large amount of configurateable things the program options dialog is subdivided into pages. You can select the current page from the tree structure on the left side of the window. The page itself will be displayed on the right side of the window.

"Easy" and "Advanced" mode

The program options dialogs knows two modes: "Easy" and "Advanced". If the "Easy" mode is activated only the really important settings will be shown. Many smaller things for tweaking the GUI will be hidden in this mode so first-time users will not get confused by the huge list of options. In the "Advanced" mode all configuration options are shown.

The mode can be changed by the "Mode" dropdown list at the bottom of the program options window. The change will take place immediately after the mode is changed. The dialog will remember the last selected mode when it is opend again later.
