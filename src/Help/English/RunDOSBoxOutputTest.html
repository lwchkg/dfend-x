DOSBox output methods test

In this window you can check the different DOSBox output methods. Not every graphics driver is supporting every DOSBox output method. If you select some not supported output method in a profile, the profile will not run.

The DOSBox output method to be used can be selected in the "Render" field on the Graphics page of the profile editor. Additionally you can change the output method for multiple or even all profiles at once in the Edit multiple profiles dialog.
