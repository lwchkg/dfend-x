Add Windows game profile

Clicking this menu item will create a new Windows game profile and open it in the profile editor dialog. Windows game profiles are not much mone than simple links to Windows executables. There no special need to start your Windows games via D-Fend Reloaded but with this option you can manage all of your games (DOS games, ScummVM games and Windows games) in one place.
