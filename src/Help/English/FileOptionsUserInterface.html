Program options dialog page "User interface"

There is only one option to change on this page:

  • Window size at program start
    In this dropdown box you can set up the appearance of the D-Fend Reloaded program window when the program starts. You can choose between "Default size", "Restore window size on program start", "Start minimized", "Start maximized" and "Fullscreen mode without window border".
    (Default value for this option: "Default size")
  • Icon sets
    From this selection box you can select the icon set to be used on the program toolbars, menus and in the dialogs. The chosen icon set will be activated as soon as you close the program options dialog.
    (Default value for this option: "Modern icon set")

The option marked with (*) is only visible in advanced mode.
