Program options dialog page "FreeDOS"

On this page you can define where the FreeDOS command line tools are located. D-Fend Reloaded can use the FreeDOS tools to offer you the usage of 4DOS as command line interpreter (via the file "4DOS.COM") and to use the alternative DOS extender DOS32A (via the file "DOS32A.EXE"). Both options can be activated by profile in the profile editor.
