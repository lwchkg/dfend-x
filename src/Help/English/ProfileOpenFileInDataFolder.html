Menu:   Profile | Open file in data folder

Open file in data folder

This menu item will open a submenu listing all file from the data folder of the selected game. By clicking one of the filenames in the submenu you can open the selected file directly. (And this will also work with files in sub-subfolders of the data folder etc.)

The data folder can be configurated on the directories page of the profile editor for the selected game.

You can also open the whole data folder in the Explorer via the "Open file in data folder" menu item.
