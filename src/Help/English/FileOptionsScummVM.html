Program options dialog page "ScummVM"

Also D-Fend Reloaded is a DOSBox frontend it can also handle Scumm based games by using ScummVM. If you enter the path to ScummVM on this page, you will be able to create ScummVM based profiles.

  • Minimize D-Fend Reloaded when ScummVM starts (*)
    If this option is activated, the D-Fend Reloaded main window will minimize as soon as you start a ScummVM based profile.
    (Default value for this option: off)
  • Restore program window when ScummVM is closed
    This option can only be activated if "Minimize D-Fend Reloaded when ScummVM starts" is active. It will restore the D-Fend Reloaded main window as soon as the ScummVM instance you have started by starting some profile closes.
    (Default value for this option: off)
  • Center ScummVM window (*)
    If this option is activated, the ScummVM program window will be centered on the screen if ScummVM is not started in fullscreen mode.
    (Default value for this option: off)
  • Hide ScummVM console (*)
    If this option is activated, the ScummVM console window will be made invisible if ScummVM is not started in fullscreen mode.
    (Default value for this option: off)
  • Additional ScummVM command line parameters (*)
    The command line parameters entered in this line will be passed to ScummVM when starting a ScummVM based profile.

The options marked with (*) are only visible in advanced mode.
