Quit

This program option closes D-Fend Reloaded. Because D-Fend Reloaded is keeping all profile files up to date on disk at any time, there is no need to save anything when closing the program.

You can also close D-Fend Reloaded at any time by just clicking on the "x" on the right of the title bar of the program window.

Background operations

D-Fend Reloaded allows to use zip-files as DOSBox drives. It will expand the zip-file before DOSBox is starting and repacking the files to the zip-file when DOSBox closes. If you try to close D-Fend Reloaded while it is waiting for one or more DOSBox instances to close and to repack the corresponding zip-file, it will warn you before it will close and will give to the option to cancel the close operation.
