Program options dialog page "DOSBox CVS features"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can activate some D-Fend Reloaded configuration options the normal DOSBox builds cannot handle. You should not activate one of this settings unless you are using a special CVS DOSBox build supporting this things.

  • Mounts editor features
    • Allow multiple floppy images per drive
      If this option is activated the mount dialog will offer to mount multiple floppy images for one virtual drive. You can swap between the images in DOSBox by pressing Ctrl+F4.
    • Activate PhysFS tab
      The Phys file system allows to mount zip files in DOSBox as drives. DOSBox will use the zip files as read only file systems and will perform all write operation into another folder. With this checkbox you can activate the PhysFS function in the mount dialog.
  • Graphics setup features
    In this box you can activate several extended graphical features in the profile editor not supported by the default DOSBox build.
  • Sound setup features
    In this box you can activate one extended sound feature in the profile editor not supported by the default DOSBox build.
  • Printer setup features
    The default DOSBox build does not support printing. If you activate the checkbox in this field, D-Fend Reloaded will offer you some printer setup options in the profile editor.
  • Network setup features
    In this box you can activate NE2000 emulation setting on the Network page of the profile editor.
