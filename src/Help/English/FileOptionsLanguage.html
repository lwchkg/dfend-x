Program options dialog page "Language"

There three dropdown boxes on this page:

  • Program language
    In this dropdown box you can select the language that will be used in the program for displaying the menus and dialogs. Changes in this box will take effect instantly without closing the dialog or restarting the program.

    To create your own language just click on "Create new language" and enter the names for the shown identifiers in your language. You do not need any external program (even not a text editor) to create a D-Fend Reloaded language file.

    To add a new (downloaded) language file to you D-Fend Reloaded installation, copy the language file to the "lang" subfolder of the D-Fend Reloaded data folder (usualy "C:\Users\YourName\D-Fend Reloaded) and reopen the program options dialog. Now you should be able to select the new language from the dropdown box.
  • DOSBox language file
    In this box you can select the language that should be used inside DOSBox for messages. D-Fend Reloaded will list all available *.lng files from your DOSBox directory. To add a new (downloaded) DOSBox language just copy the *.lng file to your DOSBox directory and reopen the program options dialog.

    The language chosen on this page will apply to the default DOSBox installation (see DOSBox page of the program options dialog).
  • Installer language
    From this dropdown box you can select the language that should be used in the installer packages. Changing this language will need to change a value in the all users part of the registry and will (under Vista) therefor ask you for admin rights to do this.
