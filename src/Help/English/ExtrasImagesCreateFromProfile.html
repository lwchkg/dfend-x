Create image file based profile from normal profile

From this Dialog you can convert a DOSBox profile in which the game is ran directly inside the DOSBox DOS emulation to a harddisk image based profile. This can be usefull if the game is not running correctly inside the DOSBox DOS emulation.

Besides the name of the profiel to transfer you only have to define how many megabyte of free space you want to have on the harddisk image for further saved games etc., if the image shall be compressed in the NTFS file system and if HIMEM and EMM386 shall be added to the image. In most cases it is a good choise to add these memory manager but some games will not run if they are loaded.

The source profile will not be changed while creating the harddisk image and the new profile. The harddisk image based new profile will appear as a seond entry in the games list.
