Templates

In the temapltes dialog you can manage the templates that can be used when creating new profiles. The templates are just profiles not specified to a special game. By clicking on "Edit" you can edit each template just like a normal profile. By clicking on "Use" (not available for auto setup templates) you can create a new profile based on the chosen template. And by clicking on "Add" you can either create a completely new template or create a new template from an existing profile.

In D-Fend Reloaded there are two kinds of templates:

Templates

The templates in this list can be used for creating new profiles via the "Add from template" menu item from the "Profile" menu.

There are two special templates in this list called "Default template" and "Default template for ScummVM profiles". This templates are displayed in blue, cannot be renamed and cannot be deleted. This templates are used when creating a new DOSBox or ScummVM profile via the "Add" or the "Add ScummVM profile" menu item of the "Profile" menu.

Auto setup templates

The templates in this listed will be used to automatically identifiy a game when adding a profile via the new profile wizard.

 

Hint:

From the "Edit" menu of the templates dialog you can also edit multiple templates at once by clicking "Edit multiple templates".
