Program options dialog page "Games list"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can tweak the visual appearance of the games list.

  • Background color / image
    Via the three radio buttons "Default background color", "Background color" and "Background image" you can set up whether D-Fend Reloaded should you the default Windows background color for the games list, a custom color or a custom image.
    (Default value for this option: "Default background color")
  • Font size and font color
    From this two fields you can choose the font size and color for the records in the games list.
    (Default value for this option: font size 9, font color "Window text")
  • Display value for "Not set"
    In this box you can define what D-Fend Reloaded should display in view mode report, if some game information are missing. This value will be displayed in the "Genre" column for example if you have not defined any genre for a game.
    (Default value for this option: "Not set")
  • Font style for favorites
    The font styles (bold, italic and/or underline) you have chosen in this box will be used for profiles you have marked as favorites.
    (Default value for this option: bold)
  • Use program file icon for Windows games profiles
    If this option is activated, the default icon from the program file will be used in the games list for Windows profiles if there is no user defined icon for the profile.
    (Default value for this option: on)
  • Shop tooltips in games list
    If activated, D-Fend Reloaded will show tooltips with additional information if you place the mouse over any game in the games list.
    (Default value for this option: on)
  • Restore last selected profile on program restart
    Reselects the last selected profile on D-Fend Reloaded start.
    (Default value for this option: off)
