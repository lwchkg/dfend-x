Program options dialog page "Service"

This page will offer you 2 to 5 (depending of the setup mode) function for maintaining your profiles database:

  • Restore "DOSBox DOS" profile
    Clicking this button will restore the default "DOSBox DOS" profile. If the profile does not exist it will be created and if it already exists it will be overwritten.
  • Restore default templates
    By clicking this button all templates will be restored to the state when D-Fend Reloaded was installed.

The following settings will only be visible if the advanced setup mode is selected at the bottom of the program options dialog:

  • Delete old unsued setup files from D-Fend v2
    D-Fend v2 (classic) used to store additional conf files in the profiles folder which are not used by D-Fend Reloaded. Additonally canceling the add profile dialog in D-Fend v2 can result in some Tempprof.* files in the profiles folder. Clicking this button will delete these old, unused files.
  • Make absolute to relative paths
    When clicking this button, D-Fend Reloaded will check all profiles for path names that are stored absolute but can be expressed as relative paths and will change them. (Using relative path names is required for usage of the functions to transfer profiles and to build installer packages.)
  • Caclulate checksums for all profiles
    D-Fend Reloaded is using checksums to monitor the game files for changes. Checksums are created (if not existent) or checked (if they already exist) when opening and closing the profile editor and when trying to run a profile. By clicking this button you can create checksums for all profiles at once.
  • Clear package cache and lists
    By clicking this option you can delete all downloaded packages lists and cached data packages. The next time you open the packages manager you will need to redownload the list files.
  • Create capture folders for all profiles
    Clicking this button creates capture folders (for screenshots, records sounds and videos) for all profiles. If there is no capture folder defined in the profile an unique folder name is derived from the profile name.
  • Create game data folders for all profiles
    Clicking this button creates game data folders (for manuals etc.) for all profiles. If there is no game data folder defined in the profile an unique folder name is derived from the profile name.
  • Rename all media files for all profiles
    Renames the screenshots, sounds or cpatures videos of all profiles to user defined file name schema.
  • Reset profiles to templates
    Clicking this button will open the Resetting profiles dialog.
