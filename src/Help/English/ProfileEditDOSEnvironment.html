Profile editor dialog page "DOS environment"

On this page you can configurate the DOS environment DOSBox will offer to the game:

  • Reported DOS version
    From this box you can select which DOS version DOSBox should report. Most programs do not care for the DOS version but if you want to run a program requiering a special DOS version you can select this version from this box.
  • Use 4DOS as command line interpreter (*)
    If this option is activated, 4DOS will be used as the command line interpreter instead of the default DOSBox command line interpreter.
  • Environment variables
    In this list you can specify the environment variables DOSBox should use. In most cases there if no need to configurate any special environment variables.

The option marked with (*) is implemented by using the FreeDOS 4DOS command line interpreter. This means FreeDOS must be available to use this options. You can configurate the path to FreeDOS on the FreeDOS page of the program options dialog. If the information text below the option is displayed in gray, FreeDOS is available. Otherwise the text will be displayed in red.
