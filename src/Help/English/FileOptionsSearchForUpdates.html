Program options dialog page "Search for updates"

On this page you can define if and how often D-Fend Reloaded should look for updates.

Program updates

The update checker will not transfer any of your private data to the internet. It will just read the content of a text file offered on the D-Fend Reloaded homepage. If the version named in this file is bigger than the current version number it will offer to you to download und run the update installer also named in this text file. That all.
(Default value for this option: "Only manual")

Add current version number to request URL

If this option is activated, the update checker will add the current version number to the request URL. This will allow the scripts on the D-Fend Reloaded homepage to offer different (maybe smaller) update packages for different versions (see details below). At the moment the update information text file is a real text file and this feature is not used on the homepage.
(Default value for this option: on)

Package lists update

From this box you can select if and when D-Fend Reloaded should update the lists offered in the packages manager. The lists can also be updated via the "Update package lists" button in the package manager dialog at any time manually.
(Default value for this option: "Only manual")

Cheats database

From this box you can select if and when D-Fend Reloaded should search and install updates to the cheats database.
(Default value for this option: "Only manual")

Mobygames reader configuration update

From this box you can select when the function for reading game information from the Mobygames homaepage should update its configuration from the D-Fend Reloaded homepage. To be able to react on changes of the Mobygames homepage the reader function is configurateable. On the D-Fend Reloaded homepage there is always a file telling the reader how to interpret the Mobygames search result page to get the game information.
(Default value for this option: "On first use each day")

Search for updates now

Clicking this button opens the same dialog as the menu item Search for updates from the help menu.

Details on the function of the program update checker

The program update checker is reading this file from Sourceforge: "http://dfendreloaded.sourceforge.net/UpdateInfo.txt" The file contains two lines; in the first line the version number of the current D-Fend Reloaded version on the server is given and in the second line the download URL of the update installer for this version is given. If the version number in the UpdateInfo.txt is bigger than the version number of the currently installed D-Fend Reloaded version, the update checker will ask you, if you want to update to the new version.

The update checker will use the windows connection settings (which you can edit in the Internet Explorer). This means: If you have setup a proxy server in the Internet Explorer, the D-Fend Reloaded update checker will use this proxy, too.
