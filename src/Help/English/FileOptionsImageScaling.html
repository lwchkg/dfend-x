Program options dialog page "Image scaling"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can setup how D-Fend Reloaded should scale images up und down if they shall be displayed at some size different to the native size of the image. The algorithm "off" will just remove lines or columns or double them. All other algorithms use more complex ways of adding or removing pixels. "off" is the fastest method but will give the worst results. All other algorithms will give better results but scaled image may will look a bit less sharp.
