Profile editor dialog page "Mouse"

On this page you can setup the virtual mouse DOSBox should emulate.

  • Auto lock mouse
    If this option is activated the mouse pointer will be locked inside the DOSBox window if you click on it. This can solve some mouse problems when running programs that uses the mouse in windows mode. To free the mouse pointer you have to press Ctrl+F10.
  • Mouse sensitivity
    By changing this value to can setup the mouse sensitivity. A higher value means the mouse will move faster and a lower values will make it move slower.
  • Force 2-button mouse mode (*)
    DOSBox emulates a virtual 3-button mouse. For most games this works very well but some games need a 2-button mouse. This the mouse moves very strange and cannot be conntrolled in the game, you should try this option.
  • Swap mouse buttons (*)
    By activating this option you can swap the left and the right mouse button inside DOSBox.

The two options marked with (*) are implemented by using the FreeDOS mouse driver. This means FreeDOS must be available to use this options. You can configurate the path to FreeDOS on the FreeDOS page of the program options dialog. If the information texts below the two options are displayed in gray, FreeDOS is available. Otherwise the texts will be displayed in red.
