Program options dialog page "Windows games"

On this page you can setup the behavior of the D-Fend Reloaded main window when a Windows game is started or is closed.

  • Minimize D-Fend Reloaded when Windows game starts
    If this option is activated, the D-Fend Reloaded main window will minimize as soon as you start a Windows based game profile.
    (Default value for this option: off)
  • Restore program window when Windows game is closed
    This option can only be activated if "Minimize D-Fend Reloaded when Windows game starts" is active. It will restore the D-Fend Reloaded main window as soon as the Windows game instance you have started by starting some profile closes.
    The main window will only be restored if the Windows program was running at least 5 seconds. So if the called program is only a launcher for the main game the D-Fend Reloaded program window will not be restored immediately.
    (Default value for this option: off)
  • Allow D-Fend Reloaded to create archive files from Windows profiles and import Windows profile package files
    If this option is on, D-Fend Reloaded will let you create archive file packages from Windows profiles and will let you import archive files containing Windows profiles. Transfering Windows games using D-Fend Reloaded archive files is not recommended and can cause many problems.
    (Default value for this option: off)
