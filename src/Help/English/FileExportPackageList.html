Menu:   File | Export | Create package list file

Package list creator

Using this dialog you can create package list files from the data in your D-Fend Reloaded system. Later you can reimport the data linked to the packages list via the Download packages function. You can also share your exported data packages to other users via the Internet. Please remark it is not allowed by law to share copyrighted content like games. Only games marked as Shareware or Freeware are allowed to share. Additionally you can always share selfmade content like D-Fend Reloaded language files, auto setup templates and selfmade icons.
