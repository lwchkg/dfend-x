Program options dialog page "Tree structure"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

This page allows you to tweak the appearance of the tree structure on the left side of the main window.

  • Restore selected games filter on restart (*)
    If this checkbox is checked, D-Fend Reloaded will reselect the last chosen filter from the tree on the left side of the program window when the program gets restarted.
    (Default value for this option: on)
  • Default background color / Background color
    Via this two radio buttons you can choose whether the tree structure should use the Windows default background color or a custom background color.
    (Default value for this option: "Default background color")
  • Font size and font color
    From this two fields you can choose the font size and color for the records in the tree structure.
    (Default value for this option: font size 9, font color "Window text")
  • Filter categories to be displayed
    In thix box you can select which default filter categories shall be displayed in the filtering tree.
  • User-defined filter categories
    The names you enter in this edit field (each name a line) will be available as categories for filtering the profiles from the tree structure (just like the default fileters "Genre", "Developer" etc. shown in the tree structure.
    (Default value for this option: none)
  • Add existing user key
    Clicking on this button will show a dropdown menu containing all available user defined keys from all profiles. By clicking on any user key from this menu will add the key to the user-defined filters list.
