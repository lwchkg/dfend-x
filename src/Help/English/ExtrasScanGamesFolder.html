Scan games folder for new games

With this function you can scan the directory defined on the directories page of the program options dialog as the games folder for new games. D-Fend Reloaded can automatically add the found games to the games list.

  • Name scheme
    From this field you can select how the profile name should be chosen when adding a game without a matching auto setup template. The game filename or the folder name can be chosen as profile name for the new profile.
