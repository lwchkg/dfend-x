Program options dialog page "Media viewers"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On this page you can define how D-Fend Reloaded should open pictured, sound files and videos for previewing. For each of the three file types you can define individually:

  • Use internal viewer
    If this option is checked D-Fend Reloaded will use its internal viewer for displaying or playing files of the selected type. The internal viewer will open very quickly and has some forward/backward buttons to cycle through the captured files of the given type but does not has any functions for editing the files.
    (This option is the default value.)
  • Use default program for opening files
    If this option is activated the media files of the selected type will be opened in the default program registered in Windows for the given file extension. Doubleclicking on any screenshot, sound or video file in D-Fend Reloaded will have the same result than doubleclicking the file in the Explorer.
  • Use custom program
    if this option is checked D-Fend Reloaded will open files of the selected type in the external program defined in the following line.

Show internal image viewer as modal window

If this option is activated, the screenshot image viewer will be displayed as a modal dialog window. This means as long as this window is visible the main window will not respond on input events.
(Default value for this option: on)
