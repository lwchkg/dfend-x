Menu:   File | Export | Export gamelist

Export gamelist

This option allows to export a list of all currently available profiles in different formats. D-Fend Reloaded offers the following file format for list export:

  • Text (*.txt):
    Create a plain text file with information about all profiles. This export mode works just like the D-Fend function for exporting game lists.
  • Table (*.csv):
    If you choose this format D-Fend Reloaded will create a table of the availalbe profiles. (Each line a profile.) You can read csv-files with nearly all spreadsheet programs.
  • HTML (*.html):
    This option will create a html-file with a table containing information about all profiles. You can use files like this directly as part of your homepage.
  • XML (*.xml):
    This option will create a xml-file.
