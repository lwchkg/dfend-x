Menu:   File | Import | Import conf file

Import conf-file

DOSBox uses conf-files for setup. This program option allows to make D-Fend Reloaded profiles any from existing DOSBox conf-file.

Importing profiles from D-Fend

If you want to import profiles from an D-Fend installation, you can import the conf-files or the prof-files. Because the prof-files contain more information it is recommended to import the prof-files.

Import via drag&drop

You can also import conf and prof files by just droping them from the Explorer to the D-Fend Reloaded program window.
