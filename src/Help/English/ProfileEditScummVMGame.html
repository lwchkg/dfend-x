Profile editor dialog page "Game specific settings"

On this page you can configurate various settings which only apply to some games. The names of the games where the settings will take effect are listed below the checkboxes and edit fields. Additionally the checkboxes and edit fields are enabled only in case the currently chosen game for the profile is supporting them.
