Program options dialog page "Default values"

This page will only be visible if the advanced setup mode is selected at the bottom of the program options dialog.

On many pages of the profile editor there are dropdown boxes with default values for DOSBox settings. On this program options page you can change the default values offered on this dropdown boxes.

The default value lists normally will cover all values DOSBox understands. So you should not change this list unless you are using a special DOSBox build with more configuration options.
