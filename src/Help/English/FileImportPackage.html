Packages manager

The packages manager offers you many data packages (games, icons for the games list, D-Fend Reloaded langauge files, icon sets fo the D-Fend Reloaded toolbars etc.) for download and direct install from the D-Fend Reloaded homepage and other sources. In the program window the available data packages are listed by types on different pages. The install button at the bottom of the window allows you to download and automatically install the selected packages with a single click.

Update lists

If you click the "Update lists" button D-Fend Reloaded will check if the package lists on the server have been changed and will download and use new lists. If you hold down the shift key will clicking on "Update lists" button, D-Fend Reloaded will download the lists from the server even if the last change date of the lists on the server indicates the lists have not been changed.

Repositories editor

By clicking the "Repositories lists" button you can add additionally download sources besides the official D-Fend Reloaded homepage as sources for data packages. If you have created data packages and a corresponding list file via the Create package list file function and archived all of them on a CD, you can add the path to the list file on the CD here. (D-Fend Reloaded will look on all installed CD drives for the data packages regardless of the selected drive letter.) You can also add Internet sources hier. Please note the added a non trusted source might bring you besided legal problems security problems. Downloading data packages, containing program code, from a not trusted source can damage your system!
