In Spielstand nach Adresse suchen

Mit Hilfe dieser Funktion können Sie nach der Adresse eines bestimmten Wertes (z.B. nach der Adresse des Wertes für das Geld im Spiel) suchen. Wenn D-Fend Reloaded eine eindeutige Adresse für einen bestimmten Wert gefunden hat, kann diese automatisch in die Cheatsdatenbank eingetragen werden, so dass Sie die Spielstände dieses Spiels fortan direkt über die Funktion Gespeicherten Spielstand ändern entsprechend ändern können.

D-Fend Reloaded bietet zwei Möglichkeiten zur Suche nach einer Adresse. Wenn Sie den genauen Zahlenwert des zu ändernden Wertes kennen, können Sie direkt nach diesem Wert suchen. Findet D-Fend Reloaded den Wert mehrfach in der Datei, so speichert es die Adresse und durch eine weitere Suche in einer anderen Spielstandsdatei können Sie Adresse weiter eingrenzen. Überlicherweise bleibt nach einigen wenigen Suchschritten nur noch eine Adresse übrig, die dann in die Datenbank eingetragen werden kann. Ist der genaue Zahlenwert jedoch unbekannt, so muss eine etwas aufwendigere Suchmethode angewandt werden. Im ersten Schritt legt D-Fend Reloaded eine Kopie der aktuellen Spielstandsdatei an. Wenn Sie den Suchdialog später erneut öffnen, können Sie angeben, ob sich der Wert, dessen Adresse gesucht ist, seit dem letzten Aufruf erhöht oder erniedrigt hat. D-Fend Reloaded vergleich dann den aktuellen Spielstand mit dem gespeicherten, alten Spielstand. Auch auf diese Weise ergibt sich eine Reihe von Adresskandidaten. Wiederum durch weitere Suchen mit "gestiegen" oder "gefallen" lässt sich die Liste weiter eingrenzen, so dass auch bei dieser Methode am Ende eine eindeutige Adresse übrig bleibt. Allerdings sind bei der Suche ohne konkreten Wert üblicherweise mehr Schritte nötig, um eine Adresse zu finden.
