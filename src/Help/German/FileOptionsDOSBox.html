Programmeinstellungen-Dialogseite "DOSBox"

Auf dieser Seite können Sie einstellen, welche DOSBox-Installationen Sie in D-Fend Reloaded verwenden möchten.

  • DOSBox-Installation
    D-Fend Reloaded kann beliebig viele DOSBox-Installationen verwalten. Über dieses Auswahlfeld können Sie einstellen, welche DOSBox-Installation Sie konfigurieren wollen. Sie können beliebig viele Installationen hinzufügen und wieder löschen. Lediglich die als "Standard" bezeichnete Installation kann nicht gelöscht werden. Sie können für jedes Profile im Profileditor einstellen, welche der hier konfigurierten DOSBox-Installationen Sie verwenden wollen.
    (Vorgabewert für diese Option: "Standard")
  • DOSBox-Verzeichnis
    In diesem Eingabefeld können Sie den Pfad zu der zu konfigurierenden DOSBox-Installation angeben.

    In der portablen Betriebsart wird dieses Verzeichnis beim Schließen des Programmes relativ zum D-Fend Reloaded Programmverzeichnis gespeichert. Wenn D-Fend Reloaded neu gestartet wird, wird das DOSBox-Verzeichnis wieder in einen absoluten Pfad umgewandelt, d.h. in diesem Eingabefeld steht immer eine absolute Pfadangabe, auch wenn der Pfad in der portablen Betriebsart korrekt relativ gespeichert wird.

"Weitere Einstellungen für die DOSBox-Installation"-Dialog

Diese Schaltfläche ist nur sichtbar, wenn unten rechts im Programmeinstellungen-Dialog der Konfigurationsmodus "Vollständig" gewählt ist.

  • DOSBox-Verzeichnis
    (Der Pfad in diesem Eingabefeld ist der selbe, wie der im Hauptfenster des Programmeinstellungen-Dialogs.)
    In diesem Eingabefeld können Sie den Pfad zu der zu konfigurierenden DOSBox-Installation angeben.

    In der portablen Betriebsart wird dieses Verzeichnis beim Schließen des Programmes relativ zum D-Fend Reloaded Programmverzeichnis gespeichert. Wenn D-Fend Reloaded neu gestartet wird, wird das DOSBox-Verzeichnis wieder in einen absoluten Pfad umgewandelt, d.h. in diesem Eingabefeld steht immer eine absolute Pfadangabe, auch wenn der Pfad in der portablen Betriebsart korrekt relativ gespeichert wird.
  • DOSBox-Sprachdatei
    In dieser Box können Sie die Sprache auswählen, die DOSBox zur Anzeige von Meldungen verwenden soll. D-Fend Reloaded zeigt in dieser Auswahlbox alle in dem DOSBox-Verzeichnis verfügbaren *.lng-Dateien an. Um neue Sprachdateien zu dieser Auswahlliste hinzuzufügen, müssen Sie nur die (neu heruntergeladene) Sprachdatei in das DOSBox-Programmverzeichnis kopieren; beim nächsten Öffnen dieses Dialogs wird sie dann in dieser Liste angezeigt.
  • Tastaturbelegung
    In dieser Box können Sie die standardmäßig mit dieser DOSBox-Version zu verwendende Tastaturbelegung auswählen. Ist hier "default" gewählt, so wird die Standard-Tastaturbelegung der gewählten Programmsprache verwendet, bzw. die Tastaturbelegung des zu startenden Profils, wenn in dem Profil explizit eine Tastaturbelegung definiert ist.
  • Zeichensatz (Codepage)
    In dieser Box können Sie die standardmäßig mit dieser DOSBox-Version zu verwendende Codepage auswählen. Ist hier "default" gewählt, so wird die Standard-Codepage der gewählten Programmsprache verwendet, bzw. die Codepage des zu startenden Profils, wenn in dem Profil explizit eine Codepage definiert ist.
  • Tastaturzuordnungsdatei (Keymapping-Datei)
    In dieser Tastaturzuordnungsdatei werden Änderungen an der Vorgabe-Tastenzuordnen, die über den Menüpunkt DOSBox-Keymapper starten des "Ausführen"-Menüs vorgenommen wurden, gespeichert. Der Pfad zu der Tastaturzuordnungsdatei kann als absoluter Pfad oder aber relativ zu dem Basisverzeichnis angegeben werden.
    (Vorgabewert für diese Option: ".\mapper.txt")
  • Zusätzliche Parameter beim Aufruf von DOSBox
    Die in diesem Eingabefeld angegebenen Parameter werden beim Aufruf von DOSBox an die Kommandozeile angehängt.
    (Vorgabewert für diese Option: keine)
  • DOSBox-Console ausblenden
    Die DOSBox-Console zeigt zusätzliche Status- und Fehler-Meldungen an. Beim Ausführen normaler Spiele gibt es eigentlich keinen Bedarf, die DOSBox-Console anzuzeigen und es sollte normalerweise also auch nicht notwendig sein, diese Option zu deaktivieren.
    (Vorgabewert für diese Option: ein)
  • DOSBox-Fenster zentrieren
    Wenn diese Option aktiviert ist, wird das DOSBox-Fenster zentriert auf dem Desktop dargestellt (solange Sie keine Profile starten, die das DOSBox-Fenster in den Vollbildmodus versetzen.)
    (Vorgabewert für diese Option: aus)
  • Screensaver deaktivieren, wenn DOSBox läuft
    Durch das Aktivieren dieser Option können Sie verhindern, dass sich der Windows-Screensaver aktiviert so lange DOSBox läuft. Ist diese Option aktiviert, so wird der Screensaver auch dann nicht aktiviert, wenn Sie keine Eingaben per Tastatur oder Maus vornehmen.
    (Vorgabewert für diese Option: aus)
  • DOSBox-Console nicht automatisch schließen, wenn ein Fehler aufgetreten ist
    Wenn diese Option aktiviert ist, wird die DOSBox-Console nicht automatisch geschlossen, wenn beim Betrieb von DOSBox ein Fehler aufgetreten ist und DOSBox beendet wurde.
    (Vorgabewert für diese Option: ein)
  • SDL Videodriver
    In diesem Auswahlfeld können Sie den SDL Videotreiber, den Sie verwenden wollen, auswählen. SDL stellt die Grafik-Basis von DOSBox dar.
    (Vorgabewert für diese Option: "DirectX")
  • Warnungen bei Fehlern beim Erzeugen der DOSBox-Pfadnamen anzeigen
    DOSBox verwendet eine andere Methode als Windows zur Erzeugung kurzer Pfadnamen aus langen Pfadnamen. Daher können lange Pfadnamen zur Verwendung innerhalb von DOSBox (z.B. in "cd"-Befehlen) nicht einfach mit Hilfe der Windows-API-Funktion in kurze 8.3-Namen umgeformt werden, sondern müssen gemäß dem DOSBox-Algorithmus abgekürzt werden. Tritt bei dieser Übersetzung eines langen Pfadnamen ein Fehler auf, so erscheint eine Warnmeldung, wenn diese Option aktiviert ist.
    (Vorgabewert für diese Option: ein)
  • Benutzereinstellungen
    Die Zeilen, die Sie in dieses Eingabefeld schreiben, werden beim Aufruf von DOSBox an die Konfigurationsdatei angehängt. Auf diese Weise können Sie beliebige Konfigurationseinstellungen überschreiben und Einstellungen vornehmen, die von der grafischen Oberfläche in D-Fend Reloaded nicht abgebildet werden.
