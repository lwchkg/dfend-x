Programmeinstellungen-Dialogseite "Nach Updates suchen"

Auf dieser Dialogseite können Sie einstellen ob und wie oft D-Fend Reloaded nach Updates suchen soll.

Programm-Updates

Die Updateüberprüfung überträgt keine persönlichen Daten ins Internet. Bei der Updateüberprüfung wird lediglich der Inhalt einer auf der D-Fend Reloaded Homepage angebotenen Textdatei heruntergeladen und ausgewertet. Wenn die Versionsnummer, die in dieser Textdatei angegeben ist, größer als die Versionsnummer der aktuellen Installation ist, bietet die Updateprüfung an, den in der zweiten Zeile der Textdatei angegebenen Updateinstaller herunter zu laden und auszuführen.
(Vorgabewert für diese Option: "Nur manuell auf Updates prüfen")

Kennung der aktuellen Version in die Abfrage-URL integrieren

Wenn diese Option aktiviert ist, fügt die Updateprüfung die Nummer der aktuell verwendeten D-Fend Reloaded Version an die Abfrage-URL für die Update-Informationsdarei an. Dadurch wird des den Skripten auf der D-Fend Reloaded Homepage ermöglicht, je nach verwendeter Version verschiedene Update-Dateien (die dann ggf. kleiner ausfallen können) anzubieten. (Siehe dazu auch der Abschnitt über die Arbeitsweise der Updatefunktion weiter unten). Momentan wird diese Funktion jedoch nicht durch die D-Fend Reloaded Homepage ausgewertet, d.h. unabhängig von dieser Einstellung für stets die selbe Update-Datei angeboten.
(Vorgabewert für diese Option: an)

Paketlisten aktualisieren

In diesem Feld kann gewählt werden, ob und wann D-Fend Reloaded die Listen der im Paketmanager angeboten Pakete aktualisieren soll. Die Listen können auch jeder Zeit über die Funktion "Paketlisten aktualisieren" im Paketmanager manuell aktualisiert werden.
(Vorgabewert für diese Option: "Nur manuell")

Cheats-Datenbank

In diesem Feld können Sie auswählen, ob D-Fend Reloaded automatisch nach Updates für die Cheats-Datenbank suchen soll.
(Vorgabewert für diese Option: "Nur manuell")

Mobygames-Downloader Konfigurationsupdate

In diesem Feld kann gewählt werden, wann die Funktion zum Auslesen von Spieleinformationen von der Mobygames-Homepage die Konfiguration neu von der D-Fend Reloaded Homepage lesen soll. Um auf Änderungen an der Mobygames-Homepage reagieren zu können, ist es notwendig, dass die Ausleseroutinen konfigurierbar sind. Auf der D-Fend Reloaded Homepage befindet sich eine Datei, die der Auslesefunktion mitteilt, wie die Mobygames-Suchanfrage-Antwortseiten auszuwerten sind um an die Spieleinformationen zu gelangen.
(Vorgabewert für diese Option: "Täglich bei der ersten Benutzung")

Jetzt nach Update suchen

Diese Schaltfläche öfnnet den Dialog, der auch über den Menüpunkt Nach Updates suchen im "Hilfe"-Menü erreichbar ist.

Dateils zur Funktionsweise der Programmupdate-Überprüfung

Der Programmupdate-Checker läd folgende Datei von der D-Fend Reloaded Hompage auf Sourceforge herunter: "http://dfendreloaded.sourceforge.net/UpdateInfo.txt". Die Datei enthält zwei Textzeilen; in der ersten Zeile steht die Versionsnummer der aktuellen D-Fend Reloaded Version und in der zweiten Zeile die Download-Adresse für den Updateinstaller, der eine existierende D-Fend Reloaded Installation auf diese Version aktualisiert. Wenn die Versionsnummer in der UpdateInfo.txt-Datei größer ist, als die Versionsnummer der momentan verwendeten D-Fend Reloaded Installation, wird Sie der Update-Checker fragen, ob Sie jetzt auf diese Version aktualisiern wollen.

Die Update-Prüfung verwendet die Windows-Verbindungseinstellung (die Sie über die Programmeinstellungen des Internet Explorers anpassen können). Dies bedeutet insbesondere, dass D-Fend Reloaded den möglicherweise im Internet Explorer konfigurierten Proxy-Server verwenden wird.
