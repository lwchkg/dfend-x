Mehrere Profile auf Werte aus Vorlagen zurücksetzen

In diesem Dialog können Sie bestimmte Einstellungen von einem oder mehreren Profilen auf die Vorgabewerte einer Vorlage zurückseten. Der Dialog untergliedert sich in drei Seiten: Auf der ersten Seite wählen Sie die Profile, deren Eintellungen Sie zurücksetzen möchten, auf der zweiten Seite wählen Sie die Einstellungen, die Sie zurücksetzen wollen und auf der dritten Seite wählen Sie aus, zu welcher Vorlage die Werte zurückgesetzt werden sollen.
