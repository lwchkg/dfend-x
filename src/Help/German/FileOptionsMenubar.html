Programmeinstellungen-Dialogseite "Menüzeile"

Diese Dialogseite ist nur sichtbar, wenn Sie den Konfigurationsmodus "Vollständig" unten im Programmeinstellungen-Dialog gewählt haben.

Auf dieser Seite können Sie einstellen, ob im D-Fend Reloaded Programmfenster eine Menüzeile angezeigt werden soll. In diesem Fall können Sie die Menüzeile über das Kontextmenü der Symbolleiste wieder aktivieren. Um das Wiederherstellen der Menüzeile nicht unmöglich zu machen, können folglich immer nur entweder die Menüzeile oder die Symbolleiste ausgeblendet werden.
