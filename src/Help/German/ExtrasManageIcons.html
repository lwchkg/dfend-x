Menü:   Extras | Icons verwalten

Icon-Manager

In diesem Dialog können Sie die Icons, die Sie in der Spieleliste verwenden wollen, konfigurieren. Über die beiden Schaltfläche "Hinzufügen" und "Löschen" können Sie Icons in die Liste aufnehmen und wieder entfernen. (Die Icons werden physikalisch in die Liste eingefügt und nicht nur als Verknüpfung eingefügt.) Die in dieser Liste vorhandenen Icons können über die Dialogseite Profil des Profileditor-Dialogs in Profile eingebunden werden. Allerdings können Sie vom Profileditor aus auch Icon-Dateien außerhalb dieser Icon-Liste für Profile verwenden.
