Profileditor-Dialogseite "CPU"

Auf dieser Dialogseite kann eingestellt werden, wie DOSBox die CPU für das auszuführende Spiel emulieren soll.

  • Emulationskern
    In diesem Feld können Sie den Typ des zu verwendenden Emulationskerns wählen:

    • Auto: DOSBox wählt den Typ des Emulationskerns automatisch.
    • Normal: DOSBox führt die CPU-Befehle Schritt für Schritt auf einem virtuellen CPU-Kern aus. Prinzipiell können alle Spiele in dieser Betriebsart ausgeführt werden, allerdings verbraucht die Schritt-für-Schritt Emulation einer kompletten CPU verhältnismäßig viel Rechenzeit, d.h. evtl. lässt sich in dieser Einstellung nicht die gewünschte ausführungsgeschwindigkeit erreichen.
    • Dynamic: DOSBox wird versuchen, die CPU-Befehle des Spiels direkt auf der realen CPU auszuführen. Diese Emulationsart ist sehr schnell, da prinzipiell die gesamte CPU-Leistung der (modernen, schnellen) realen CPU zur Ausführung des DOS-Programms zur Verfügung steht. Allerdings kann diese direkte Abbildung auf die reale CPU bei einigen Spielen Probleme verursachen. So lange keine Probleme bei einem Spiel auftreten, sollte dieser Emulationstyp verwendet werden.
    • Simple: DOSBox führt die CPU-Befehle, genauso wie im Modus "Normal" Schritt für Schritt auf einem virtuellen CPU-Kern aus. Allerdings ist der virtuelle CPU-Kern in diesem Modus einfacher aufgebaut und damit schneller.
  • CPU Zyklen
    In dieser Box können Sie einstellen, wie viele Befehle DOSBox pro Millisekunde zu emulieren versuchen soll. Wenn Sie den Wert "Max" wählen, wird DOSBox versuchen, so viele Befehle wie möglich pro Millisekunde auszuführen. Die Einstellung "Auto" führt dazu, dass DOSBox automatisch den optimalen Wert an CPU Zyklen zu bestimmen versucht und ggf. automatisch in den Modus "Max" wechselt.
  • CPU Zyklen erhöhen
    In diesem Eingabefeld können Sie einstellen, um wie viele emulierte CPU Befehle pro Millisekunde der Zyklen-Wert erhöht werden soll, wenn Sie in DOSBox Strg+F12 drücken.
  • CPU Zyklen verringern
    In diesem Eingabefeld können Sie einstellen, um wie viele emulierte CPU Befehle pro Millisekunde der Zyklen-Wert verringert werden soll, wenn Sie in DOSBox Strg+F11 drücken.
  • CPU-Typ (*)
    Über diese Auswahlbox können Sie festlegen, welchen CPU-Typ DOSBox emulieren soll.

Die mit (*) markierte Optionen ist nur sichtbar, wenn Sie das zugehörige DOSBox CVS-Feature auf der Dialogseite DOSBox CVS-Features des Programmeinstellungen-Dialogs aktiviert haben. Die offizelle DOSBox 0.72 Version, die mit D-Fend Reloaded ausgeliefert wird, unterstützt diese zusätzliche Option nicht.
