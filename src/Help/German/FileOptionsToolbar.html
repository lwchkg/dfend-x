Programmeinstellungen-Dialogseite "Symbolleiste"

Diese Dialogseite ist nur sichtbar, wenn Sie den Konfigurationsmodus "Vollständig" unten im Programmeinstellungen-Dialog gewählt haben.

Auf dieser Seite können Sie das Erscheinungsbild und die Funktionsweise der Symbolleiste im D-Fend Reloaded Hauptprogrammfenster konfigurieren.

  • Symbolleiste anzeigen
    Durch das ein- und ausschalten dieser Option kann die gesamte Symbolleiste im Hauptfenster sichtbar oder unsichtbar gemacht werden. Wenn die Symbolleiste durch das Deaktivieren dieser Option ausgeblendet ist, haben alle weiteren Einstellungen auf dieser Seite keine weitere Bedeutung.
    (Vorgabewert für diese Option: ein)
  • Sichtbare Schaltflächen
    In diesem Listenfeld können Sie die Schaltflächen, die in der Symbolleiste sichtbar sein sollen, auswählen.
    (Vorgabewert für diese Option: Alle außer "Schließen" und "Setup")
  • Beschriftungen in der Symbolleiste anzeigen
    Wenn diese Option aktiviert wird, werden Icons und Namen auf den Schaltflächen in der Symbolleiste angezeigt. Andernfalls werden nur die Symbole ohne Beschriftungen dargestellt.
    (Vorgabewert für diese Option: ein)
  • Funktion der Hinzufügen-Schaltfläche
    Über diese Auswahlbox kann bestimmt werden, welche Funktion die "Hinzufügen"-Schaltfläche in der Symbolleiste haben soll. Zur Auswahl stehen:

    • "Hinzufügen-Dialog": Öffnet den normaen Profileditor zum Hinzufügen eines neuen Profils.
    • "Assistent": Öffnet den Assistenten zum Hinzufügen eines neuen Profils.
    • "Auswahlenü": Öffnet direkt an der Stelle der "Hinzufügen"-Schaltfläche beim Anklicken eben dieser ein Menü, aus dem die gewünschte Funktion ausgewählt werden kann.

    (Vorgabewert für diese Option: "Auswahlenü")
  • Hintergrundbild
    In diese Eingabezeile können Sie den Namen eines Bildes eingeben, welches als Hintergrundbild für die Symbolleiste verwendet werden soll.
    (Vorgabewert für diese Option: kein Hintergrundbild)
  • Schriftgröße
    In diesem Feld können Sie die Schriftgröße, die für die Beschriftungen der Schaltflächen in der Symbolleiste verwendet werden sollen, einstellen.
    (Vorgabewert für diese Option: 9)
