Erststart-Assistent

Der Erststart-Assistent hilft Ihnen bei Vornehmen der wichtigten Einstellungen unmittelbar nach der Installation von D-Fend Reloaded.

Programmsprache und DOSBox-Sprache

Die Programm- und DOSBox-Sprache wird vom Installationsprogramm übernommen, so dass diese in den meisten Fällen bereits korrekt eingestellt sein sollte.

Standard DOSBox-Installation

In diesem Feld können Sie angeben, welche DOSBox-Installation standardmäßig zum Aufruf der DOS-Spiele verwendet werden soll. Sofern Sie bei der Installation von D-Fend Reloaded die Option, DOSBox mit zu installieren, nicht explizit abgewählt haben, können Sie die Einstellung "Mit D-Fend Reloaded installierte Version" beibehalten. Unabhängig von der hier gewählten Standard-Installation können Sie später jedem Profil auch eine individuelle DOSBox-Version zuweisen.

Automatisch nach Updates suchen

Über dieses Feld können Sie festlegen, ob D-Fend Reloaded automatisch in regelmäßigen Abständen nach Programmupdates suchen soll. (Es wird dabei nur nach Updates gesucht; wenn ein Update gefunden wurde, werden Sie gefragt, ob Sie dieses installieren wollen. Eine automatische Installtion von Updates ohne Rückfrage erfolgt keinesfalls.) Wenn Sie die Vorgabeeinstellungen beibehalten, wird D-Fend Reloaded niemals automatisch Verbindungen mit dem Internet aufnehmen. Sie können auch jederzeit manuell über den Menüpunkt Nach Updates suchen im "Hilfe"-Menü einen Update-Suchvorgang starten.

Alle hier angebotenen Einstellungen können Sie auch später jederzeit in dem Programmeinstellungen-Dialog anpassen.
