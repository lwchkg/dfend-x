Conf-Dateien erstellen

Die conf-Dateien verwenden von DOSBox zum Ablegen der Konfigurationseinstellungen verwendet. Die meisten DOSBox Frontends können conf-Dateien importieren, d.h. wenn Sie mit D-Fend Reloaded erstellte Profile zu anderen Frontends übertragen wollen, so sollte dies mit conf-Dateien mit einer hohen Wahrscheinlichkeit funktionieren. Allerdings enthalten conf-Dateien weniger Informationen als die D-Fend Reloaded prof-Profildateien. Daher sollten Sie, sofern das andere Frontend D-Fend bzw. D-Fend Reloaded Profiledateien lesen kann, bevorzugt prof-Dateien zum Übertragen der Profile verwenden.

Wenn Sie sich einfach nur den Inhalt einer conf-Datei zu einem bestimmten Profil ansehen wollen, so müssen Sie dafür nicht extra diese Exportfunktion verwenden, sondern können direkt die entsprechende DOSBox conf-Datei anzeigen Funktion aus dem "Profil"-Menü verwenden.
