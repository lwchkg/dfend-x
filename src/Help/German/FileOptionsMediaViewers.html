Programmeinstellungen-Dialogseite "Medienbetrachter"

Diese Dialogseite ist nur sichtbar, wenn Sie den Konfigurationsmodus "Vollständig" unten im Programmeinstellungen-Dialog gewählt haben.

Auf dieser Dialogseite können Sie angeben, wie D-Fend Reloaded Bilder, Sounddateien und Videos zur Ansicht öffnen soll. Für jeden der drei Medientypen können Sie das Verhalten individuell festlegen:

  • Internen Betrachter verwenden
    Wenn diese Option aktiviert ist, wird D-Fend Reloaded den internen Betrachter bzw. das interne Abspielprogramm für den gewählten Medientyp verwenden. Der interne Betrachter hat den Vorteil, dass er schneller startet als ein externes Programm, außerdem können Sie bequem zwischen mehreren aufgezeichneten Dateien hin- und herschalten. Allerdings besitzen die internen Betrachter keinerlei Funktionen zum Bearbeiten von Dateien.
    (Diese Option ist der Vorgabewert.)
  • Standardprogramm zum Öffnen der Dateien verwenden
    Wenn diese Option aktiviert ist, werden Mediendateien des gewählten Typs jeweils in dem Programm, welches jeweils unter Windows als Vorgabeprogramm für den jeweiligen Dateityp eingetragen ist, geöffnet. Auf einen Screenshot, eine Sounddatei oder ein Video doppelt zu klicken hat in diesem Fall die selbe Wirkung, wie ein Doppelklick im Windows Explorer auf die entsprechende Datei.
  • Benutzerdefiniertes Programm verwenden
    Wenn diese Option in D-Fend Reloaded ausgewählt ist, werden Dateien des entsprechenden Typs in dem hier angegebenen Programm geöffnet. line.

Internen Betrachter als modalen Dialog anzeigen

Wenn diese Option aktiviert ist, wird der Screenshot-Betrachter als Dialogfenster angezeigt, d.h. so lange das Fenster geöffnet ist, kann mit dem Hauptfenster nicht weitergearbeitet werden.
(Vorgabewert für diese Option: aktiviert)
