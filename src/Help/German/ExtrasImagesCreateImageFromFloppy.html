IMG-Image von Diskette erstellen

In diesem Dialog können Sie IMG-Imagedateien von physikalischen Disketten erstellen. IMG-Diskettenimages können über die Dialogseite Laufwerke des Profileditor-Dialogs als Laufwerke in DOSBox eingebunden werden.

Über den Menüpunkt "IMG-Image auf Diskette schreiben" des "Imagedateien"-Untermenüs des "Extras"-Menü können bestehende IMG-Images wieder zurück auf Diskette geschrieben werden.
