Profileditor-Dialogseite "Hardware"

Auf dieser Dialogseite können Sie Einstellungen zur Verwendung der vorhandenen Hardware durch ScummVM vornehmen. Bei DOSBox-Profilen ist diese Dialogseite leer.

  • CD-Laufwerk zum Abspielen von Musik
    In diesem Feld können Sie einstellen, welches physikalische CD-Laufwerk Sie verwenden wollen, um von ihm CD-Audio-Musik als Hintergrundmusik im Spiel abzuspielen. Das erste CD-Laufwerk im System hat die Nummer 0.
  • Nummer des zu verwendenen Joysticks
    Wenn das Spiel, das Sie konfigurieren wollen, Joystick-Eingaben unterstützt, können Sie in diesem Eingabefeld festlegen, welchen physikalischen Joystick Sie in dem Spiel verwenden wollen. Der erste mit dem Computer verbundene Joystick bekommt von ScummVM die Nummer 0.
  • Plattform
    Normalerweise sollte die Einstellung "auto" beibehalten werden. In diesem Fall versucht ScummVM die Plattform der Spieledateien automatisch zu bestimmen. Nur wenn dieses nicht funktioniert, sollten Sie hier explizit die Plattform, für die die Spieledateien erstellt wurden, angeben.
