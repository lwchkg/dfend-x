Profileditor-Dialogseite "Serielle Schnittstellen"

Auf dieser Dialogseite können Sie die vier virtuellen seriellen Schnittstellen, die DOSBox emulieren kann, konfigurieren. Jede der vier virtuellen seriellen Schnittstellen kann in einem der folgenden Betriebsmodi arbeiten:

  • Deaktiviert
    Die entsprechende serielle Schnittstelle wird deaktiviert.
  • Dummy
    Betreibt die serielle Schnittstelle im Dummy-Modus, d.h. Sie existiert, hat aber keine weitere Funktion.
  • Modem
    Ist diese Option gewählt, so wird ein virtuelles Modem, welches mit der jeweiligen Schnittstelle verbunden ist, emuliert. Die tatsächliche Verbindung zu der Gegenstelle erfolgt über das Netzwerk des realen Computers. Dafür müssen Sie auf dem Server "Auf eingehende Verbindungen warten" wählen.
  • Nullmodem
    Ist diese Option gewählt, so wird ein virtuelles Nullmodemkabel, welches mit der jeweiligen Schnittstelle verbunden ist, emuliert. Die tatsächliche Verbindung zu der anderen DOSBox-Instanz erfolgt über das reale Netzwerk des Computers. Es muss jeweils ein Comuter als Server und einer als Client konfiguriert werden. Auf der Client-Seite muss außerdem die IP-Adresse des Servers eingetragen werden.
  • Physikalische Schnittstelle
    Mit Hilfe dieser Option kann eine virtuelle serielle Schnittstelle in DOSBox mit einer physikalischen seriellen Schnittstelle des realen Computers verbunden werden.
