Programme übertragen

Von diesem Dialog aus können Sie Profile von einer D-Fend Reloaded Installation zu einer anderen übertragen (z.B. von Ihrer normalen Installation zu einer portablen Installation auf einem USB-Stick). Bei der Übertragung eines Profils werden alle zu dem Spiel gehörenden Dateien (Profildatei, Spieleprogrammdateien, aufgezeichnete Screenshots usw. sowie die zusätzlichen Datendateien) übertragen, d.h. nach der Übertragung kann das Spiel von der Ziel-Installation aus direkt gestartet werden, ohne dass dabei die Dateien Quell-Installation verfügbar sein müssen.

Diese Funktion steht nur für Profile, die ausschließlich relative Pfade verwenden, zur Verfügung, da bei der Verwendung von absoluten Pfaden nicht klar ist, wie diese Dateien von der Quell- zu der Ziel-Installation übertragen werden sollen.

Portable D-Fend Reloaded Installation im Zielverzeichnis erstellen

Wenn diese Option deaktiviert ist, muss in dem als Zielverzeichnis angegebenen Ordner bereits eine D-Fend Reloaded Installation vorhanden sein, andernfalls wird das Übertragen der Programme scheitern. Wenn diese Option jedoch aktiviert ist, wird vor dem Übertragen der Programme eine portable D-Fend Reloaded Installation in dem Zielverzeichnis erstellt. Dafür werden die Programmdateien des Ausgangsinstallation in das Zielverzeichnis kopiert und die Konfigurationsdateien entsprechend angepasst.
