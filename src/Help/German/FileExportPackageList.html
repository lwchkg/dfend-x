Paketliste erstellen

Mit Hilfe dieser Funktion können Sie eine Paketliste für Daten, die in Ihrem D-Fend Reloaded System vorhanden sind, erstellen. Mit Hilfe solcher Paketlisten können Sie später über die Funktion Datenpakete herunterladen die exportierten Daten wieder importieren. Sie können allerdings die Paketliste (zusammen mit den zugehörigen Paketdateien) auch im Internet für andere Nutzer verfügbar machen. Bitte beachten Sie hierbei, dass keine urheberrechtlich geschützten Daten angeboten werden dürfen. Dies sind insbesondere alle Spiele, die nicht explizit als Share- oder Freeware freigegeben wurden. Selbst erstellte D-Fend Reloaded Sprachdateien, AutoSetup-Vorlegen und Icons können hingegen ohne Bedenken freigegeben werden.
