Programmeinstellungen-Dialogseite "Sicherheit"

Diese Dialogseite ist nur sichtbar, wenn Sie den Konfigurationsmodus "Vollständig" unten im Programmeinstellungen-Dialog gewählt haben.

Auf dieser Seite können die Schutzmaßnahmen gegen das unababsichtigte Löschen falscher Dateien sowie zum Schutz der einzelnen Programmdateien konfiguriert werden:

  • Vor dem Löschen von Einträgen nachfragen
    Wenn diese Option aktiviert ist, zeigt D-Fend Reloaded einen Nachfrage/Bestätigungsdialog an, wenn Sie in der Spieleliste ein Profil auswählen und Entf drücken. Ist diese Option deaktiviert, so wird das Profil sofort ohne Rückfrage gelöscht.
    (Vorgabewert für diese Option: ein)
  • Beim Deinstallieren von Spielen niemals Dateien außerhalb des Basisordners löschen
    Solange diese Option aktiviert ist, wird D-Fend Reloaded beim Deinstallieren von Spielen niemals Dateien außerhalb des Basisordners löschen. Wenn diese Option aktiviert ist, kann auch beim Deinstallieren von fehlerhaft konfigurierten Spielen keinen Schaden an der normalen Softwareinstallation entstehen. Wenn diese Option jedoch deaktiviert ist und Sie ein Spiel deinstallieren wollen, welches z.B. C:\Windows als seinen Spieleordner angibt, so können Sie damit ggf. Ihr gesamtes Betriebssystem löschen.
    (Vorgabewert für diese Option: ein)
  • DOS-Programmdateien per Prüfsumme auf Änderungen überprüfen
    D-Fend Reloaded kann in jedem Profile Prüfsummen zu der Spiele- und der Setup-Programmdatei ablegen. Bei jedem Start von Spiel oder Setupprogramm wird dann die gespeicherte Prüfsumme mit der neuberechneten Prüfsumme der Datei verglichen. Wenn die gespeicherte Prüfsumme nicht mehr zu der Programmdatei passt (und diese Option aktiviert ist), gibt D-Fend Reloaded eine Warning aus. Mit Hilfe dieser Funktion können Änderungen an Programmdateien, die evtl. durch einen Virus verursacht wurden, erkannt werden und es kann erkannt werden, wenn Sie versuchen ein unpassendes Profil (mit ggf. unpassenden Einstellungen) auf ein anderes Spiel anzuwenden.
    (Vorgabewert für diese Option: ein)
  • Dateien beim Löschen in den Papierkorb verschieben
    In diesm Feld können Sie auswählen, wann D-Fend Reloaded zu löschende Dateien in den Papierkorb verschieben soll und wann es die Dateien sofort endgültig löschen soll.
    (Vorgabewert: Alle nicht temprären Dateien in den Papierkorb verschieben)
  • Im Profileditor anbieten, DOSBox usw. mit Admin-Rechten auszuführen
    Ist diese Option aktiviert, so erscheint im Profileditor eine Auswahlbox, die es ermöglicht, DOSBox, ScummVM bzw. das gewählte Windows-Programm mit Administrator-Rechten zu starten. Normalerweise sollte dies nicht erforderlich sein. Das Starten eines Anwendungsprogramms mit Administrator-Rechten erhöht nur unötig die Gefahr, die Softwareinstallation des Systeme zu beschädigen. Die Option hier abgewählt, so wird die Funktion im Profileditor nicht nur nicht angeboten, sondern es werden Programme, auch wenn dies vorher aktiviert wurde, nicht mit Admin-Rechten gestartet.
    (Vorgabewert für diese Option: aus)

Um das Risiko, dass durch Fehlbedienungen oder Programmfehler Schäden an Ihrer Softwareinstallation auftreten, sollten sie die ersten drei Optionen stets aktiviert lassen.
