Programmeinstellungen-Dialogseite "Spieleliste"

Diese Dialogseite ist nur sichtbar, wenn Sie den Konfigurationsmodus "Vollständig" unten im Programmeinstellungen-Dialog gewählt haben.

Auf dieser Dialogseite können Sie das Aussehen der Spieleliste an ihre Wünsche anpassen.

  • Hintergrundfarbe / Hintergrundbild
    Über die drei Optionen "Standard-Hintergrundfarbe", "Hintergrundfarbe" und "Hintergrundbild" kann eingestellt werden, ob D-Fend Reloaded die von Windows vorgegebene Standardhintergrundfarbe für die Spieleliste verwenden soll, die hier angegebene benutzerdefinierte Farbe verwenden soll oder eine Bilddatei als Hintergrund der Spieleliste anzeigen soll.
    (Vorgabewert für diese Option: "Standard-Hintergrundfarbe")
  • Schriftgröße und Schriftfarbe
    Über diese beiden Felder kann sowohl die Größe als auch die Farbe der Texte in der Spieleliste eingestellt werden.
    (Vorgabewert für diese Option: Schriftgröße 9, Schriftfarbe "Windowstext")
  • Anzeigewert für "Nicht zugeordnet"
    In diesem Auswahlfeld können Sie bestimmen, was D-Fend Reloaded in der Spieleliste im Ansichtsmodus Details in den Spalten anzeigen soll, für die keine Werte vorhanden sind. (Also was z.B. in der Spalte "Genre" angezeigt werden soll, wenn für ein Spiel kein Genre definiert ist.
    (Vorgabewert für diese Option: "Nicht zugeordnet")
  • Darstellung der Favoriten
    Der Schriftart-Stil (fett, kursiv und/oder unterstrichen) den sie in diesem Feld angegeben haben, wird verwendet um die Profile, die sie als Favoriten markiert haben in der Spieleliste gegenüber den anderen Profilen hervorzuheben.
    (Vorgabewert für diese Option: fett)
  • Bei Windows-Profile Icon aus Programdatei verwenden
    Ist diese Option aktiviert, so zeigt D-Fend Reloaded in der Spieleliste bei allen Windows-Profilen das Icon aus der zu startenden Windows-Exe-Datei an, sofern für das Profil nicht explizit ein anderes Icon gewählt wurde.
    (Vorgabewert für diese Option: aktiviert)
  • Tooltips in der Spieleliste anzeigen
    Wenn dieses Option aktiviert ist, werden in der Spieleliste zusätzlich Informationen in Form einen kleinen Tooltip-Fensters angezeigt, so bald Sie den Mauszeiger über einem der Profile plazieren.
    (Vorgabewert für diese Option: aktiviert)
  • Zuletzt ausgewähltes Profile beim Programmstart wieder auswählen
    Wählt das zuletzt aktive Profile beim nächsten D-Fend Reloaded Start direkt wieder aus.
    (Vorgabewert für diese Option: deaktiviert)
