Menü:   Extras | Verknüpfungen für mehrere Profile gleichzeitig erstellen

Verknüpfungen erstellen

Mit Hilfe dieser Funktion können Sie ein Verknüpfungen für die gewählten Profile auf dem Desktop oder im Startmenü erstellen. Wenn Sie diese Verknüpfungen später anklicken, wird das entsprechende Profil automatisch gestartet.

Wenn Sie D-Fend Reloaded in einer Wine-Umgebung unrer Linux einsetzen, können Sie diese Funktion so umkonfigurieren, dass sie statt Windows-Verknüpfungen Linux Shell-Skripte erstellt. (Sie dafür die Dialogseite Wine-Unterstützung des Programmeinstellungen-Dialogs.)
