Spieleliste exportieren

Diese Funktion erlaubt es, eine Liste aller momentan verfügbaren Profile in verschiedenen Ausgabeformaten zu exportieren. D-Fend Reloaded bietet dabei in dem Speichern-Dialog folgende Formate zum Export an:

  • Text (*.txt):
    Erstellt eine einfache Textdatei mit Informationen über alle Profile. Dieses Exportformat entspricht dem Format, in dem D-Fend seine Profilliste exportieren konnte.
  • Tabelle (*.csv):
    Wenn Sie dieses Exportformat wählen, erstellt D-Fend Reloaded eine Tabelle mit allen verfügbaren Profilen. (In jeder Zeile steht ein Profil.) Die erstellten csv-Dateien können von nahezu jeder Tabellenkalkulation eingelesen werden.
  • HTML (*.html):
    In dieser Einstellung wird eine html-Datei erstellt, die intern eine Tabelle mit den Profileinformationen enthält. Eine auf diese Art exportierte Spieleliste können Sie direkt als Teil Ihrer Homepage verwenden.
  • XML (*.xml):
    In dieser Einstellung wird eine x,ö-Datei erstellt.
