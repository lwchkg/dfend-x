Profileditor-Dialogseite "MIDI"

Auf dieser Dialogseite können Sie das virtuelle, von DOSBox emulierte MIDI-Ausgabegerät konfigurieren. So lange Sie mit dem verwendeten Spiel keine Probleme bei der MIDI-Ausgabe haben, sollten Sie hier keine Veränderungen vornehmen. Insbesondere sollte in dem Auswahlfeld "MIDI-Gerät" stets "default" oder "win32" eingestellt bleiben.
