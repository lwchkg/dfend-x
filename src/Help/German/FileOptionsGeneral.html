Programmeinstellungen-Dialogseite "Allgemein"

Auf dieser Seite gibt es zwei Optionen, die aktiviert oder deaktiviert werden können:

  • In den Benachrichtigungsbereich minimieren
    Wenn diese Option aktiviert ist, wird das D-Fend Reloaded Programmfenster im minimierten Zustand nicht in der Taskleiste angezeigt, sondern nur als kleines Symbol im Benachrichtigungsbereich unmittelbar links neben der Uhrzeit ganz unten rechts auf dem Bildschirm.
    (Vorgabewert für diese Option: deaktiviert)
  • Mit Windows starten (*)
    Wenn diese Funktion aktiviert ist, wird D-Fend Reloaded bei jedem Windows-Start automatisch mitgestartet.
    (Vorgabewert für diese Option: deaktiviert)
  • Nur eine laufende Instanz des Programms zulassen (*)
    Ist diese Option aktiviert, so kann immer nur eine Instanz von D-Fend Reloaded zurzeit laufen. Wird eine weitere Instanz des Programmes gestartet, so wird lediglich die bereits laufende Instanz von D-Fend Reloaded in den Vordergrund geholt.
    (Vorgabewert für diese Option: aktiviert)

Die mit (*) gekennzeichneten Optionen sind nur sichtbar, wenn der Konfigurationsmodus "Vollständig" gewählt ist.
