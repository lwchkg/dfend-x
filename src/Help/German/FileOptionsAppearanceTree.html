Programmeinstellungen-Dialogseite "Baumstruktur"

Diese Dialogseite ist nur sichtbar, wenn Sie den Konfigurationsmodus "Vollständig" unten im Programmeinstellungen-Dialog gewählt haben.

Auf dieser Dialogseite kann das Aussehen der Kategorien-Baumstruktur auf der linken Seiten des Hauptfenster angepasst werden.

  • Zuletzt ausgewählte Kategorie beim Start erneut auswählen (*)
    Ist diese Option aktiviert, so wird die zuletzt in der Baumstruktur auf der linken Seite des Programmfensters ausgewählte Kategorie beim nächsten Start von D-Fend Reloaded sofort erneut wieder ausgewählt.
    (Vorgabewert für diese Option: aktiviert)
  • Standard-Hintergrundfarbe / Hintergrundfarbe
    Über diese beiden Auswahlknöpfe kann eingestellt werden, ob D-Fend Reloaded als Hintergrund für die Kategorienliste die Windows Standardhintergrundfarbe oder eine hier angegebene benutzerdefinierte Hintergrundfarbe verwenden soll.
    (Vorgabewert für diese Option: "Standard-Hintergrundfarbe")
  • Schriftgröße und Schriftfarbe
    Über diese beiden Felder kann sowohl die Größe als auch die Farbe der Texte in der Baumstruktur eingestellt werden.
    (Vorgabewert für diese Option: Schriftgröße 9, Schriftfarbe "Windowstext")
  • Anzuzeigende Kategorien
    In diesem Auswahlfeld kann festgelegt werden, welche der Vorgabe-Filter-Kriterien in der Baumstruktur angezeigt werden solln.
    (Vorgabewert für diese Option: alle Einträge aktiviert)
  • Benutzerdefinierte Kategorien
    Die Namen, die Sie in dieses Eingabefeld eintragen (jeweils ein Name pro Zeile) werden in der Baumstruktur als weitere Kategorien zum Filtern der Spiele in der Spieleliste angeboten (genauso wie die Filter "Genre", "Entwickler" usw.)
    (Vorgabewert für diese Option: keine)
  • Existierende Benutzerschlüssel
    Wenn Sie auf diese Schaltfläche klicken, wird eine Liste aller in den Spielen vorkommenden benutzerdefinierten Kategorienamen als auswahlmenü angezeigt. Durch das Anklicken eines solchen Kategorienamens wird dieser in die Liste der benutzerdefinierte Kategorien aufgenommen.
