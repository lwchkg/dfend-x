Programmeinstellungen-Dialogseite "ScummVM"

Auch wenn D-Fend Reloaded in erster Linie ein DOSBox-Frontend ist, kann es doch dennoch Scumm-basierende Spiele mit Hilfe von ScummVM starten. Wenn Sie auf dieser Dialogseite den Pfad zu ScummVM angeben, können Sie über den Menüpunkt ScummVM-Profil hinzufügen im "Profil"-Menü ScummVM-basierende Profile zur Spieleliste hinzufügen.

  • D-Fend Reloaded minimieren, wenn ScummVM gestartet wird (*)
    Wenn diese Option aktiviert ist, wird das D-Fend Reloaded Hauptfenster minimiert, sobald ein ScummVM-basierendes Profil gestartet wird.
    (Vorgabewert für diese Option: aus)
  • Programmfender wiederherstellen, wenn ScummVM beendet wird
    Diese Funktion kann nur aktiviert werden, wenn die Option "D-Fend Reloaded minimieren, wenn ScummVM gestartet wird" ebenfalls aktiviert ist. Diese Option stellt das D-Fend Reloaded Hauptfenster wieder her, sobald die ScummVM-Instanz, durch deren Start das Fenster minimiert wurde, wieder geschlossen wurde.
    (Voragebwert für diese Option: aus)
  • ScummVM-Fenster zentrieren (*)
    Wenn diese Option aktiviert ist, wird das ScummVM-Fenster, wenn ScummVM nicht im Vollbildmodus gestartet wird, zentriert auf dem Bildschirm angezeigt.
    (Vorgabewert für diese Option: aus)
  • ScummVM-Console ausblenden (*)
    Wenn diese Option aktiviert ist, wird das ScummVM-Consolenfenster ausgeblendet, wenn ScummVM nicht im Vollbildmodus gestartet wird.
    (Vorgabewert für diese Option: aus)
  • Zusätzliche ScummVM Aufrufparameter (*)
    Über diese Eingabezeile können zusätzliche Aufrufparameter an ScummVM beim Aufruf eines ScummVM-basierenden Profils übergeben werden.

Die mit (*) markierten Optionen sind nur sichtbar, wenn unten rechts im Programmeinstellungen-Dialog der Konfigurationsmodus "Vollständig" gewählt ist.
