Mehrere Spiele deinstallieren

In diesem Dialog können Sie mehrer Spiele gleichzeitig deinstallieren. Dies bedeutet, dass bei der Verwendung dieses Dialogs nicht nur die Profildateien zu den Spielen entfernt werden, sondern alle zu dem Spiel gehörenden Dateien gelöscht werden. Wenn Sie hingegen nur die D-Fend Reloaded Profildateien löschen wollen und die eigentlichen Spiele behalten wollen, müssen Sie die Funktion "Löschen" aus dem "Profil"-Menü verwenden.

Aktionen

In diesem Feld können Sie auswählen, welche Dateien gelöscht werden sollen. Wenn Sie die gewählten Spiele vollständig löschen wollen, können Sie den Vorgabewert "Profil-Listen Eintrag und alle Programmdatenverzeichnisse löschen" ausgewählt lassen. D-Fend Reloaded löscht grundsätzlich nur Dateien, die von keinem anderen Profil verwendet werden. Außerdem werden nur Dateien gelöscht, die sich innerhalb des Basisordners und seinen Unterordnern befinden (siehe Dialogseite Verzeichnisse des Programeinstellungen-Dialogs). Dadurch wird sichergestellt, dass das Deinstallieren von falsch konfigurierten Spielen trotzdem zu keinen Schäden an der Softwareinstallation außerhalb des D-Fend Reloaded Basisverzeichnisses führen kann. (Sie können diese Schutzfunktion auf der Dialogseite Sicherheit des Programeinstellungen-Dialogs deaktivieren, allerdings wird dringend empfohlen, diese Schutzfunktion aktiviert zu lassen.)
