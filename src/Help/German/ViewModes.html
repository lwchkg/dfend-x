Ansichtsmodi

Über die sieben folgenden Einträge des "Ansicht"-Menüs können Sie das Aussehen der Spieleliste konfigurieren:

  • Einfache Liste (keine Icons):
    Zeigt nur die Namen der Profile an. Die Profile werden spaltenweise angeordnet.
  • Einfache Liste:
    Zeigt nur die Namen der Profile und die zugehörigen (kleinen) Icons an. Die Profile werden spaltenweise angeordnet.
  • Details (keine Icons):
    Zeigt eine Vielzahl an Informationen über jedes Profil (Name, Genre, Entwickler usw.) an. Sie können auf der Dialog seite Spalten in der Spieleliste des Programmeinstellungen-Dialogs bestimmen, welche Spalten angezeigt werden sollen. Die Profile werdne in insgesamt einer Spalte untereinander angeordnet.
  • Details:
    ´ Zeigt eine Vielzahl an Informationen über jedes Profil (Name, Genre, Entwickler usw.) inklusive des Profil-Icons an. Sie können auf der Dialog seite Spalten in der Spieleliste des Programmeinstellungen-Dialogs bestimmen, welche Spalten angezeigt werden sollen. Die Profile werdne in insgesamt einer Spalte untereinander angeordnet.
  • Kleine Icons:
    Zeigt nur die Namen der Profile und die zugehörigen (kleinen) Icons an. Die Profile werden zeilenweise angeordnet.
  • Icons:
    Zeigt nur die Namen der Profile und die zugehörigen (großen) Icons an. Die Profile werden zeilenweise angeordnet.
  • Screenshots:
    Zeigt die aufgezeichneten Screenshots als ganz große Icons zu den Profilen an. Die Screenshots werden zeilenweise angeordnet. (Siehe auch die Dialogseite Liste im Screenshots-Modus im Programmeinstellungen-Dialog.)

 

Hinweis:

Sie können den Screenshot, der in der Spieleliste im Ansichtsmodus "Screenshots" angezeigt werden soll über das Kontextmenü der Screenshotsvorschau am unteren Rand des Programmfesnters auswählen.
