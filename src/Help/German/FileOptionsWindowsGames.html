Programmeinstellungen-Dialogseite "Windows-Spiele"

Auf dieser Seite können Sie das Verhalten des D-Fend Reloaded Programmfensters beim Start und beim Beenden eines Windows-Spiels festlegen.

  • D-Fend Reloaded minimieren, wenn ein Windows-Spiel gestartet wird
    Wenn diese Option aktiviert ist, wird das D-Fend Reloaded Hauptfenster minimiert, sobald ein Windows-Spiel Profil gestartet wird.
    (Vorgabewert für diese Option: aus)
  • Programmfender wiederherstellen, wenn das Windows-Spiel beendet wird
    Diese Funktion kann nur aktiviert werden, wenn die Option "D-Fend Reloaded minimieren, wenn ein Windows-Spiel gestartet wird" ebenfalls aktiviert ist. Diese Option stellt das D-Fend Reloaded Hauptfenster wieder her, sobald das Windows-Spiel, durch deren Start das Fenster minimiert wurde, wieder geschlossen wurde.
    Das Programmfenster wird nur dann wiederhergestellt, wenn das gestartete Windows-Programm mindestens 5 Sekunden gelaufen ist. Auf dieser Weise wird verhindert, dass das D-Fend Reloaded Programmfenster wiederhergestellt wird, wenn es sich bei der gestarteten Datei nur um ein Start-Programm für das eigentliche Spiel handelt.
    (Voragebwert für diese Option: aus)
  • Archive von Window-Profilen erstellen und Windows-Profil-Pakete installieren
    Wenn diese Option aktiviert ist, können Archivedateien von Windows-basierenden Profilen erstellt werden und es können Archivdateien, die Windows-basierende Profile enthalten, importiert werden. Das Übertragen von Windows-basierenden Spielen in Form von D-Fend Reloaded Archivdateien kann vielfältige Probleme verursachen und wird daher nicht empfohlen.
    (Voragebwert für diese Option: aus)
