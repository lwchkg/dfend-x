Cheats-Datenbank bearbeiten

In diesem Dialog können Sie die Cheats Datenbank von D-Fend Reloaded bearbeiten. Sie können bestehende Cheats bearbeiten oder löschen und neue Datensätze hinzufügen.

In der obenen Hälfte befindet sich eine Liste mit Spielen und bei dem jewiligen Spiel möglichen Änderungsaktionen. Aktionen können z.B. "Mehr Geld" oder "Mehr Trefferpunkte für alle Charaktere" sein. Jede Aktion kann wiederum aus mehreren Schritten bestehen. Jede Änderung eines Wertes an einer Adresse ist ein Schritt einer Aktion.

In der unteren Hälfte des Dialogs können Sie die Daten zu dem gewählten Schritt der gewählten Aktion angeben. Es gibt im Wesentlichen zwei Arten von Operationen, die D-Fend Reloaded ausführen kann. Zum einen können feste Werte an bestimmte Adressen geschrieben werden und zum anderen kann der Benutzer gefragt werden, welcher Wert an eine bestimmte Adresse geschrieben werden soll. In den Feldern für Werte und Adressen können Zahlen im Dezimal- und auch im Hexadezimalsystem eingegeben werden. Um anzudeuten, dass es sich bei einer Zahl um einen Hexadezimalwert handelt, muss der eigentlich Zahl ein "$" vorangestellt werden.

Die hier konfigurierten Cheats können über die Funktion Gespeicherten Spielstand ändern abgerufen werden. Soforn Sie noch nicht wissen, an welche Adresse in dem gespeicherten Spielstand sich der Wert befindet, den Sie ändern möchten, so können Sie die Funktion zum Suchen von Adressen verwenden.
