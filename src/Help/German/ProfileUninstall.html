Profil deinstallieren

Über diesen Menüpunkt kann das gewählte Profile inklusiver aller zugehörigen Dateien deinstalliert werden. Sie können dabei auswählen, welche Komponenten des Profils entfernt werden sollen (Programmdateien, vom Benutzer angelegte Datendateien, gespeicherte Screenshots usw.).

Aus dem Deinstallieren-Dialog, der erscheint wenn Sie diesen Menüpunkt anklicken, können Sie auswählen welche Dateien konkret gelöscht werden sollen. Wenn Sie ein Spiel vollstöndig entfernen wollen, können Sie alle Optionen in diesem Dialog ausgewählt lassen. D-Fend Reloaded bietet nur Dateien zum Deinstallieren an, die nicht von anderen Profilen verwendet werden und die außerdem innerhalb des Basisordners und seinen Unterordnern liegen (siehe Verzeichnisse Seite im Programmeinstellungen-Dialog). Dadurch wird verhindert, dass das Deinstallieren von falsch konfigurierten Profilen andere Spiele oder gar Dateien außerhalb des D-Fend Reloaded Basisordners beschädigen kann. (Sie können diese standardmäßig aktivierte Schutzfunktion auf der Dialogseite Sicherheit des Programmeinstellungen-Dialogs deaktivieren, wenn Sie dies wirklich wollen.)
