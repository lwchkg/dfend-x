Profileditor-Dialogseite "Netzwerk"

Auf dieser Dialogseite können Sie einstellen, welche Netzwerk-Funktionen DOSBox den Programmen, die in DOSBox laufen, zur Verfügung stellen soll und wie diese auf das Netzwerk des realen Computers abgebildet werden sollen.

  • UDP/IP Emulation aktivieren
    Über dieses Auswahlfeld können Sie die gesamte Netzwerk-Unterstützung in DOSBox ein- oder ausschalten. Sind die Netzwerkfunktionen in DOSBox deaktiviert, so haben alle weiteren Optionen auf dieser Dialogseite keine Auswirkungen.
  • IPX Verbindung herstellen
    In diesem Feld können Sie einstellen, ob der jeweilige Computer als Client (initiiert Verbindungen) oder als Server (wartet auf eingehende Verbindungen) auftreten soll.
  • IPX Server-IP-Adresse (für Client-Modus)
    In dieses Eingabefeld können Sie die Adresse des Server eingeben, wenn Sie in dem "IPX Verbindung herstellen"-Feld die Betriebsart "Client" gewählt haben. Der Client versucht dann unter der angegebenen Adresse Kontakt mit dem Server aufzunehmen.
  • IPX Port
    In diesem Eingabefeld können Sie die Portnummer angeben, die DOSBox zum Verbindungsaufbau verwenden soll. Die Portnummer sollte auf Client und Server identisch eingestellt sein.
