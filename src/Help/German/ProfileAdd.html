DOSBox Profil hinzufügen

Durch das Anklicken dieses Menüpunktes können Sie ein neues DOSBox-Profil in D-Fend Reloaded anlegen. Das neu erstellte Profil wird sofort im Profileeditor-Dialog geöffnet. (Wenn Sie eingestellt haben, dass sie den klassischen Profileditor-Dialog verwenden möchten, so wird dieser geöffnet; andernfalls öffnet sicht der moderne Profileditor-Dialog.)

Wenn Sie ein ScummVM-basierendes Profil oder ein Profil für ein Windows-Spiel hinzufügen möchten, so können Sie dies über die Menüpunkte ScummVM-profil hinzufügen und Windows-Spiel hinzufügen aus dem "Profil"-Menu.
