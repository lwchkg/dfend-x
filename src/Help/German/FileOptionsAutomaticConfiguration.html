Programmeinstellungen-Dialogseite "Automatische Konfiguration"

Diese Dialogseite ist nur sichtbar, wenn Sie den Konfigurationsmodus "Vollständig" unten im Programmeinstellungen-Dialog gewählt haben.

Auf dieser Seite können Sie einstellen, wann D-Fend Reloaded ein neues Spiel vollautomatisch konfigurieren soll und wann es nach Benutzereinstellungen fragen soll.

  • Archivdatei Import: Ohne Rückfrage importieren, wenn passende AutoSetup-Vorlage existiert
    Wenn diese Option aktiv ist, zeigt D-Fend Reloaded beim Importieren von Archivdateien (über den Menüpunkt via Datei|Importieren|Archivdateien oder via Drag&Drop) keinen Dialog zu Auswahl der Vorlage und zur Wahl des Namen des Spiels an, wenn dieses über eine AutoSetup-Vorlage identifiziert werden konnte. Gibt es keine passende AutoSetup-Vorlage für das Spiel, so wird der Auswahldialog auf jeden Fall angezeigt.
    (Vorgabewert für diese Option: an)
  • Funktionsweise des Assistenten
    In diesem Feld können Sie eingestellen, wie viele Fragen der Assistent beim Hinzufügen von Spielen stellen soll. Konkret kann eingestellt werden, wann die Dialogseiten zur Auswahl einer Vorlage und zur Eingabe der Meta-Informationen zu einem Spiel angezeigt werden sollen. Es kann eingestellt werden, dass diese Seiten immer, nur wenn das Spiel nicht über eine AutoSetup-Vorlage identifiziert werden kann oder nie angezeigt werden sollen.
    (Vorgabewert für diese Option: "Spiel automatisch konfigurieren, wenn passende AutoSetup-Vorlage existiert")
  • Dateinamen zur Identifikation von Spielen, die installiert werden müssen
    Wenn sich in einer zu importierenden Archivdatei lediglich eine Programmdatei mit einem der in diesem Feld angegebenen Namen befindet, so schlägt D-Fend Reloaded vor, das Spiel mit Hilfe der Installationsunterstützung zu installieren.
