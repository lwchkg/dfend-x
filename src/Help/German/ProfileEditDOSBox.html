Profileditor-Dialogseite "DOSBox-Einstellungen"

Auf dieser Seite können Sie DOSBox-spezifische Einstellungen vornehmen.

  • DOSBox-Priorität (Vordergrund)
    Über dieses Auswahlfeld können Sie die Priorität, die Windows DOSBox gibt, wenn es im Vordergrund läuft, einstellen. Windowsprogramme erhalten standardmäßig die Priorität "normal". Wenn Sie DOSBox eine höhere Priorität geben, bekommt es mehr CPU-Leistung zugewiesen, als andere gleichzeitig laufende Programme.
  • DOSBox-Priorität (Hintergrund)
    Über dieses Auswahlfeld können Sie die Priorität, die Windows DOSBox gibt, wenn es im Hintergrund läuft (d.h. wenn ein anderens Windowsprogramm gerade den Eingabefokus besitzt), einstellen. Windowsprogramme erhalten standardmäßig die Priorität "normal". Wenn Sie DOSBox eine höhere Priorität geben, bekommt es mehr CPU-Leistung zugewiesen, als andere gleichzeitig laufende Programme.
  • DOSBox beim Programmende schließen
    Wenn diese Option aktiviert ist, wird das DOSBox-Fenster geschlossen, sobald das Programm, welches Sie in DOSBox ausführen wollen, beendet wird. Wenn die Option deaktiviert ist, bleibt das DOSBox-Fenster geöffnet und zeigt die DOS-Kommandozeile an. In diesem Fall müssen Sie DOSBox manuell durch die Eingabe von "exit" oder in dem Sie auf den Schließenknopf oben rechts im DOSBox-Fenster klicken, beenden.
    Wenn sich das Programm, dass Sie ausführen wollen, sofort wieder schließt und dabei noch eine Fehlermeldung anzeigt, so ist es empfehlenswert, diese Option zu deaktivieren, da sich andernfalls das DOSBox-Fenster sofort wieder schließt und Sie keine Gelegenheit haben, die Meldung zu lesen. Auch wenn Sie mit der Kommandozeile arbeiten wollen (und evtl. gar kein voreingestelltes Programm ausführen wollen), müssen Sie diese Option deaktivieren.
  • Vorgabe DOSBox-Installation verwenden
    In dem Auswahlfeld unter dieser Option können Sie die DOSBox-Installation, die Sie zum Ausführen des Spiels verwenden wollen, angeben. In diesem Auswahlfeld werden alle auf der Dialogseite DOSBox des Programmeinstellungen-Dialogs angegebenen DOSBox-Installationen aufgelistet.
  • Diese benutzerdefinierte DOSBox-Installation verwenden
    Alternativ zu der Wahl einer vorgegebenen DOSBox-Installation können Sie auch in diesem Eingabefeld den Pfad zu einer beliebigen weiteren DOSBox-Installation, die beim Aufruf des Spiels verwendet werden soll, angeben.
  • Benutzerdefinierte Einstellungen für die DOSBox-Console
    Durch das Aktivieren dieser Option können Sie die auf der DOSBox Dialogseite des Programmeinstellungen-Dialogs festgelegte Einstellung dazu, ob die DOSBox-Console angezeigt wird überschreiben und festlegen, dass für das aktuelle Profile die Console stets angezeigt oder stets ausgeblendet werden soll.
  • Benutzerdefinierte Sprache für DOSBox verwenden Durch das Aktivieren dieser Option können Sie die auf der DOSBox Dialogseite des Programmeinstellungen-Dialogs festgelegte Sprache für Meldung in DOSBox außer Kraft setzen und eine benutzerdefinierte Sprache für von DOSBox auszugebende Meldungen festlegen.
  • Benutzereinstellungen
    Die Zeilen, die Sie in dieses Eingabefeld schreiben, werden beim Aufruf von DOSBox an die Konfigurationsdatei angehängt. Auf diese Weise können Sie beliebige Konfigurationseinstellungen überschreiben und Einstellungen vornehmen, die von der grafischen Oberfläche in D-Fend Reloaded nicht abgebildet werden.
