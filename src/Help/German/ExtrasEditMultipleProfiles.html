Mehrere Profile gleichzeitig bearbeiten

In diesem Dialog können Sie Änderungen an mehreren Profile gleichzeitig vornehmen. Der Dialog besteht aus drei Seiten; auf der ersten Seite können Sie die zu bearbeitenden Profile auswählen und auf der zweiten und dritten Seite können Sie angeben, welche Datenfelder auf jeweils welchen Wert geändert werden sollen. Wenn Sie auf die "Ok"-Schaltfläche klciekn, werden die gewählten Änderungen an den gewählten Profilen vorgenommen.
