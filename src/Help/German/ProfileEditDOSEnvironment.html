Profileditor-Dialogseite "DOS-Umgebung"

Auf dieser Dialogseite können Sie die DOS-Umgebung, die DOSBox den Spielen zur Verfügung stellen soll, konfigurieren:

  • Gemeldete DOS-Version
    In diesem Eingabefeld können Sie die DOS-Version, die DOSBox melden soll, einstellen. Die meisten Programme fragen diese Version nicht ab und laufen mit jeder DOS-Version. Einige Programme setzen jedoch eine bestimmte DOS-Version für den Betrieb voraus. Sollte das Programm, das Sie ausführen wollen mit der Meldung, dass die DOS-Version unpassend wäre, abbrechen, so können Sie die passende DOS-Version in dieses Eingabefeld eintragen.
  • 4DOS als Kommandozeilen-Interpreter verwenden (*)
    Wenn diese Option aktiviert ist, wird 4DOS statt des normalen DOSBox-Kommandozeilen-Interpreters verwendet.
  • Umgebungsvariablen
    In dieser Tabelle können Sie beliebige Umgebungsvarialben angeben, die DOSBox den Programmen zur Verfügung stellen soll. In den meisten Fällen sollten zur korrekten Ausführung des Spiels hier keine weiteren Einträge nötig sein. (Die häufig benötigte Blaster-Umgebungsvariable, die die Konfigurationsdaten der SoundBlaster-Karte angibt, wird von DOSBox beim Start automatisch gesetzt.)

Die mit (*) markierte Option wird über den mit FreeDOS mitgelieferten 4DOS-Kommandozeilen-Interpreter realisiert. Dies bedeutet, dass diese Option nur dann zur Verfügung steht, wenn Sie auf der FreeDOS Seite des Programmeinstellungen-Dialogs den Pfad zu Ihrer FreeDOS-Installation angegeben haben. Der Informationstext unter dieser Option wird in grau dargestellt, wenn FreeDOS verfügbar ist. Andernfalls wird der Text in rot angezeigt.
