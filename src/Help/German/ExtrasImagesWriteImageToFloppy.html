IMG-Image auf Diskette schreiben

Von diesem Dialog aus können Sie IMG-Imagedateien auf physikalische Disketten schreiben.

IMG-Diskettenimages können von physikalischen Disketten über den Menüpunkt "IMG-Image von Diskette erstellen" aus dem "Imagedateien"-Untermenü des "Extras"-Menü erstellt werden.

IMG-Diskettenimages können über die Dialogseite Laufwerke des Profileditor-Dialogs als Laufwerke eingebunden werden.
