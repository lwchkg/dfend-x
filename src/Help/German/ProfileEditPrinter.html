Profileditor-Dialogseite "Drucker"

Auf dieser Dialogseite können Sie die Drucker-Emulation in DOSBox konfigurieren.

Diese Dialogseite ist nur dann sichtbar, wenn die Drucker-Emulation auf der Seite DOSBox CVS-Features des Programmeinstellungen-Dialogs aktiviert ist. Die offizielle DOSBox Version 0.72, die mit D-Fend Reloaded ausgeliefert wird, unterstützt die Drucker-Emulation nicht.

  • Drucker-Emulation aktivieren
    Über diese Option kann die gesamte Drucker-Emulation ein- und ausgeschaltet werden. Wenn die Drucker-Emulation deaktiviert ist, sind alle weiteren Einstellungen auf dieser Dialogseite ohne weitere Bedeutung.
  • Auflösung (in DPI)
    Über dieses Eingabefeld können Sie die Auflösung (gemessen in dot per inch, Punkte pro Zoll) des virtuellen Druckers festlegen.
  • Papierbreite and Papierhöhe
    Über diese beiden Eingabefelder können Sie die größe des virtuellen Papiers einstellen. Die Größenangaben werden in der Einheit 1/10 Zoll gemessen.
  • Ausgabeformatformat
    In diesem Auswahlfeld kann festgelegt werden, in welchem Format die Ausgaben, die auf dem virtuellen Drucker ausgegeben werden, gespeichert werden sollen:

    • png:Erzeugt eine png (portable network file) Grafikdatei mit der Druckerausgabe. DOSBox fragt bei Bedarf nach einem Dateinamen für die Ausgabedatei.
    • ps:Erzeugt eine ps (post script) Datei mit der Druckerausgabe. DOSBox fragt bei Bedarf nach einem Dateinamen für die Ausgabedatei.
    • bmp:Erzeugt eine bmp (bitmap) Grafikdatei mit der Druckerausgabe. DOSBox fragt bei Bedarf nach einem Dateinamen für die Ausgabedatei.
    • printer:Sender die Ausgabe auf dem virtuellen Drucker direkt an den realen Windows-Standarddrucker.
  • Mehrseiten-Ausdrucke aktivieren
    (Diese Option ist nur für das Ausgabeformat Postscript von Bedeutung.) Wenn diese Option deaktiviert ist, wird jede Seite, die auf dem virtuellen Drucker in DOSBox ausgegeben wird, in einer eigenen Datei abgelegt. Wenn diese Option aktiviert ist, werden alle Seiten in eine Datei geschrieben. Um die Ausgabe zu beenden und die Ausgabedatei abzuschließen, müssen Sie in DOSBox Strg+F2 drücken.
