Programmeinstellungen-Dialogseite "Vorgabewerte"

Diese Dialogseite ist nur sichtbar, wenn Sie den Konfigurationsmodus "Vollständig" unten im Programmeinstellungen-Dialog gewählt haben.

Auf vielen Dialogseiten des Profileditors befinden sich Auswahlfelder, in denen zwischen verschiedenen, von DOSBox unterstützten Werten für eine Eigenschaft gewählt werden kann. Auf dieser Programmeinstellungen-Seite können Sie konfigurieren, welche Werte in den jeweiligen Auswahlfeldern angeboten werden sollen.

Die Vorgabeeinstellungen für diese Werte decken alle von DOSBox unterstützten Werte ab. Daher sollten Sie in diesen Listen nur dann Werte hinzufügen, wenn Sie eine spezielle DOSBox-Version verwenden, die zusätzliche Werte für bestimmte Eigenschaften unterstützt.
