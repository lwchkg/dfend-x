Profileditor-Dialogseite "Start"

On this page you can setup what DOSBox should do when it starts (that means you can setup here the way DOSBox will start the chosen game).

  • Spiel-Start überspringen
    Wenn diese Option aktiviert ist, werden die auf der Profil-Seite eingetragenen Programm- und Setupdateien ignoriert und DOSBox ohne den Aufruf dieser Dateien gestartet.
  • Mounting überspringen
    Wenn diese Option aktiviert ist, werden die Einstellungen, die Sie auf der Dialogseite Laufwerke vorgenommen haben, ignoriert und es werden beim Start von DOSBox keine Laufwerke eingebunden.
  • Autoexec.bat
    Die DOS-Befehle, die Sie in diesem Eingabefeld eintragen, werden beim Aufruf von DOSBox unmittelbar vor dem Aufruf des Spiels ausgeführt, d.h. den Befehlen in diesem Eingabefeld kommt eine ähnliche Bedeutung zu, wie den Befehlen inder Autoexec.bat-Datei in einer realen DOS-Umgebung.
  • Abschluss
    Die DOS-Befehle, die Sie in diesem Eingabefeld eintragen, werden nach dem Ende des Spiels ausgeführt und können z.B. Aufräumarbeiten ausführen.
  • DOSBox normal starten
    Wenn diese Option gewählt ist, wird das Spiel bzw. die Setupdatei, die Sie auf der Dialogseite Profil angegeben haben, normal gestartet. Diese Einstellung sollten Sie so lange Sie keine alternativen Betriebssysteme in Imagedateien ausführen wollen, stets aktiviert lassen.
  • Festplattenimage starten
    Wenn Sie ein bootfähiges Festplattenimage erstellt haben und dieses auf der Laufwerke-Seite dieses Dialogs so konfiguriert haben, dass es ohne Laufwerksbuchstaben eingebunden wird, können Sie hier einstellen, dass von diesem Image gestartet werden soll.
  • Diskettenimage starten
    Wenn Sie von einem Diskettenimage in DOSBox booten wollen, müssen Sie diese Option aktivieren. Diskettenimages müssen im Gegensatz zu Festplattenimages nichtals Laufwerke eingebunden sein, um von ihnen starten zu können. Sie müssen lediglich hier die Dateinamen der zu verwendenden Diskettenimages angeben. Das erste Image wird zum Booten verwenden. Über die Tastenkombination Strg+F4 können Sie zwischen den virtuellen Disketten wechseln während DOSBox läuft.
