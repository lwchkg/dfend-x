DOSBox conf-Datei anzeigen

Wenn Sie diesen Menüpunkt anklicken, wird für das gewählte Profil eine temporäre DOSBox Konfigurationdatei erstellt und in einem Texteditor geöffnet.

Wenn Sie Probleme mit dem Ausführen von einzelnen Spielen haben und im einem Internet-Forum um Hilfe bitten wollen, so sollten Sie diese confDatei mit posten, da sie alle Informationen, die beim Aufrufen an DOSBox übergeben werden, enthält und so eine sehr gute Basis für die Untersuchung von Problemen darstellt.
