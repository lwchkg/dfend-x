Profileditor-Dialogseite "ScummVM-Einstellungen"

Auf dieser Dialogseite können Sie ScummVM-spezifische Einstellungen vornehmen. Abhängig von dem auf der Dialogseite Profil gewählten Spiel, sind einige Einstellungen, die für das jeweilige Spiel nicht unterstützt werden, deaktiviert.

  • Sprache
    In diesem Auswahlfeld können Sie die Sprache, die Sie in dem Spiel verwenden wollen, auswählen. Die Option "Standard" bedeutet, dass keine Sprachinformationen beim Aufruf von ScummVM übermittelt werden. (Diese Option wird immer dann angeboten, wenn laut der Liste in D-Fend Reloaded für das vorliegende Spiel in ScummVM keine verschiedenen Sprachen unterstützt werden.) Über die Option "Benutzerdefiniert" können Sie frei eine beliebige Sprache festlegen, auch wenn diese laut Liste nicht unterstützt wird. Sie können jedoch auch die Liste der jeweils unterstützten Sprachen direkt über die Seite Voragebwerte des Programmeinstellungen-Dialogs verändern und ggf. ergänzen.
  • Autospeicher-Intervall
    In diesem Eingabefeld können Sie die Anzahl an Sekunden zwischen zwei Autospeicher-Vorgängen festlegen.
  • Text-Geschwindigkeit
    In diesem Eingabefeld können Sie die Anzeigegeschwindigkeit für Texte einstellen.
  • Untertitel aktivieren (in Spielen mit Sprachausgabe)
    Wenn diese Option aktiviert ist, werden in Spielen, die über eine Sprachausgabe verfügen, Untertitel angezeigt. (In Spielen ohne Sprachausgabe werden die Dialogtexte unabhängig von dieser Einstellung stets angezeigt.)
  • Nachfrage vor Programmende anzeigen
    Wenn diese Option aktiviert ist, zeigt ScummVM einen Rückfragedialog vor dem Programmende an.
  • Verzeichnis für Spielstände
    In diesem Feld können Sie einstellen, ob ScummVM die Spielstände in dem Spieleverzeichnis oder in einem benutzerdefinierten Verzeichnis ablegen wollen.
  • Extraverzeichnis für zusätzliche Spieledaten
    In diesem Verzeichnis können weitere, zu dem Spiel gehörende Datendateien abgelegt werden. (Bitte sehen Sie in der ScummVM-Dokumentation nach, für welche Versionen werlcher Spiele dieses Extraverzeichnis verwendet werden kann.)
  • Zusätliche ScummVM Aufrufparameter
    Die hier angegebenen Parameter werden zusätzlich beim Start des Spiels an ScummVM übergeben.
  • Benutzereinstellungen
    Die Zeilen, die Sie in dieses Eingabefeld schreiben, werden beim Aufruf von ScummVM an die Konfigurationsdatei angehängt. Auf diese Weise können Sie beliebige Konfigurationseinstellungen überschreiben und Einstellungen vornehmen, die von der grafischen Oberfläche in D-Fend Reloaded nicht abgebildet werden.
