Menüzeile anzeigen

Wenn Sie diese Option deaktivieren, wird im D-Fend Reloaded Programmfenster keine Menüzeile mehr angezeigt. In diesem Fall können Sie die Menüzeile über das Kontextmenü der Symbolleiste wieder aktivieren. Um das Wiederherstellen der Menüzeile nicht unmöglich zu machen, können folglich immer nur entweder die Menüzeile oder die Symbolleiste ausgeblendet werden.
