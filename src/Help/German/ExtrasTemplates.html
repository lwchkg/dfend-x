Vorlagen

In dem Vorlagendialog können Sie die Vorlagen, auf deren Basis neue Profile erstellt werden können, verwalten. Bei den Vorlagen handelt es sich um Profile, die zu keinem speziellen Spiel zugeordnet sind. Über die Schaltfläche "Bearbeiten" in diesem Dialog können Sie die Vorlagen genauso bearbeiten, wie normale Profile. Über die "Verwenden"-Schaltfläche können Sie eine Vorlage verwenden, um von ihr ein neues Profil zu stellen. (Die Funktion "Verwenden" steht für Autosetup-Vorlagen nicht zur Verfügung.) Über die "Hinzufügen"-Schaltfläche können Sie neue Vorlagen anlegen und auch Vorlagen von bestehenden Profilen erstellen.

In D-Fend Reloaded gibt es zwei Arten von Vorlagen:

Vorlagen

Die Vorlagen aus dieser Liste können bei der Erstellung neuer Profile über den "Hinzufügen von Vorlage"-Menüpunkt aus dem "Profil"-Menü verwendet werden.

In dieser Liste gibt es zwei spezielle Vorlagen die mit "Standardvorlage" und Standardvorlage für ScummVM-Profile" beschriftet sind. Diese Vorlagennamen werden in blau dargestellt und können nicht gelöscht oder umbeannt werden. Diese beiden Vorlagen werden beim Erstellen von Profilen über die Menüpunkte "Hinzufügen" bzw. "ScummVM-Profile hinzufügen" aus dem "Profil"-Menü verwendet.

Autosetup-Vorlagen

Die Vorlagen, die in dieser Liste angezeigt werden, werden zur automatischen Konfiguration von Spielen bei der Verwendung der Funktion "Hinzufügen mit Assistent" aus dem "Profil"-Menü verwendet.

 

Hinweis:

Über den Menüpunkt "Mehrere Vorlagen gleichzeitig bearbeiten" des "Bearbeiten"-Menüs des Vorlagendialogs können Sie auch mehrere Vorlagen gleichzeitig bearbeiten.
