Profileditor-Dialogseite "Laufwerke"

Auf dieser Seite können Sie die virtuellen Laufwerke, die DOSBox den Programmen, die in DOSBox laufen sollen, zur Verfügung stellen soll, konfigurieren. DOSBox kann physikalische Laufwerke, einzelne Ordner, Imagedateien und (wenn Sie ein neueres CVS Build von DOSBox einsetzen) Zip-Dateien als Laufwerke einbinden. Außerdem lassen sich mit Hilfe von D-Fend Reloaded auch mit der normalen DOSBox-Versin Zip-Dateien als Laufwerke verwenden. (D-Fend Reloaded packt die Zip-Datei dabei vor dem Start von DOSBox aus und nach dem Beenden von DOSBox wieder ein.)

Über die Schaltenflächen unten auf dieser Dialogseite können Sie virtuelle Laufwerke hinzufügen, bearbeiten und löschen. Außerdem können Sie auch Laufwerke hinzufügen, in dem Sie in der Laufwerksliste Einfg drücken, Laufwerke bearbeiten, in dem Sie Enter drücken und Laufwerke löschen, in dem Sie Entf drücken.

Sie können D-Fend Reloaded auch anweisen, die benötigten Laufwerke für das Spiel automatisch zu erstellen, in dem Sie auf die Schaltfläche "Autom. erstellen" klicken. In diesem Fall wird ein virtuelles Laufwerk so als Verknüpfung mit einem physikalischen Ordner (bevorzugt dem Spieleordner) erstellt, dass DOSBox die Spieledateien erreichen kann. (Achtung: Evtl. durch das Spiel benötigte Verknüpfungen zu CD-Laufwerken usw. können nicht erkannt werden und folglich auch nicht automatisch erstellt werden.)

  • Zusätzlich alle aktuell verfügbaren CDs automatisch mounten
    Wenn diese Funktion aktiviert ist, bindet D-Fend Reloaded zusätzlich zu den explizit in der oben stehenden Liste angegebenen Laufwerken auch noch alle CD-Laufwerke beim DOSBox-Start ein.
  • Einbinden von Laufwerken nach dem DOSBox-Start verhindern
    Wenn dieser Sicherheitsmodus aktiv ist, können Sie (bzw. Programme, die in DOSBox laufen) nach dem Programmstart von DOSBox keine weiteren Laufwerke mehr einbinden. Wenn diese Option eingeschaltet ist, können Sie also sicher sein, dass selbst ein Virus nicht Zugriff auf Daten, die Sie nicht hier explizit eingebunden haben, erlangen kann.

Der "Laufwerk bearbeiten"-Dialog

In diesem Dialog können Sie den Typ des Laufwerk, das DOSBox emulieren soll, einstellen und dann Einstellungen für den gewählten Typ vornehmen. Aus der Auswahlfeld "Typ" können Sie einen der folgenden Laufwerkstypen wählen:

  • Ordner als Festplatte
    Diese Option erlaubt es ein beliebiges Verzeichnis als Festplatte in DOSBox einzubinden. (Achtung: Sie sollten nicht versuchen das Wurzelverzeichnis Ihrer Festplatte als Festplatte in DOSBox einzubinden. Dies hätte zwar den Vorteil, dass Ihnen in DOSBox alle Ordner der realen Festplatte zur Verfügung stünden und auch die Struktur identisch wäre, allerdings hätten damit eben auch alle Programme innerhalb von DOSBox vollen Zugriff auf alle Verzeichnisse der realen Festplatte und können so Ihre gesamte Softwareinstallation beschädigen. Wenn Sie hingegen nur einen Spieleordner als Festplatte einbinden, so können bösartige Programme bzw. Fehlbedienungen innerhalb von DOSBox keinen Schaden an Dateien außerhalb dieses Ordner anrichten.)
  • Ordner als Diskette
    Über diese Option können Sie ein beliebiges Verzeichnis als virtuelles Diskettenlaufwerk in DOSBox einbinden.
  • CD-Laufwerk
    Diese Option erlaubt es, ein reales CD-Laufwerk als virtuelles CD-Laufwerk in DOSBox einzubinden. Sie haben dabei mehrere Möglichkeiten, nach der das Laufwerk beim Start von DOSBox ausgewählt werden soll. U.a. ist es möglich, das zu verwendende Laufwerk an Hand der eingelegten CD automatisch auszuwählen. Wenn Sie einen festen Pfad als CD-Laufwerk mounten wollen, so sollte der Pfad auf das Wurzelverzeichnis eines realen CD-Laufwerks verweisen.
  • Diskettenimage
    Diese Option erlaubt es, ein Diskettenimage als virtuelles Diskettenlaufwerk in DOSBox einzubinden. Über die Schaltfläche "Imagedatei erstellen" auf der rechten Seite des Dialogs können Sie ein neues Diskettenimage von einer realen Diskette erstellen oder auch ein neues, leeres Diskettenimage erstellen.
  • ISO-Image
    Diese Option erlaubt es, ein ISO-Image einer CD als virtuelles CD-ROM-Laufwerk einzubinden. Über die Schaltfläche "Imagedatei erstellen" auf der rechten Seite des Dialogs können Sie ISO-Images von realen CDs auslesen.
  • Festplattenimage als Festplatte
    Diese Option erlaubt es, ein Festplattenimage als virtuelle Festplatte in DOSBox einzubinden. Über die Schaltfläche "Imagedatei erstellen" auf der rechten Seite des Dialogs können Sie neue, leere Festplattenimages erstellen.
  • DOSBox PhysFS (*)
    Über diese Option können Sie eine Zip-Datei als Laufwerk in DOSBox einbinden. DOSBox erstellt das Laufwerk intern aus der Zip-Datei und einem Verzeichnis; es ließt dabei aus der Zip-Datei und führt alle Schreiboperationen in das angegebene Verzeichnis durch (d.h. die Zip-Datei wird von DOSBox nicht verändert.) Nach dem Programmende von DOSBox wird der Inhalt des Verzeichnisses optional in die Zip-Datei eingepackt.
  • Zip-Datei
    Über diese Option können Sie eine Zip-Datei als Laufwerk in DOSBox einbinden, ohna dass dafür die DOSBox PhysFS-Unterstützung benötigt wird (welche in der normalen DOSBox-Version auch nicht vorhanden ist). D-Fend Reloaded entpackt dafür den Inhalt der Zip-Datei vor dem Start von DOSBox in ein Verzeichnis und bindet dieses in DOSBox als Laufwerk ein. Nach dem Programmende von DOSBox wird der Inhalt des Verzeichnisses optional wieder in die Zip-Datei eingepackt.

Die mit (*) markierte Option ist nur verfügbar, wenn die zugehörige Option auf der Dialogseite DOSBox CVS-Features des Programmeinstellungen-Dialogs aktiviert ist. Die offizielle DOSBox Version 0.73, die mit D-Fend Reloaded ausgeliefert wird, unterstützt diesen Mount-Typ und den Sicherheitsmodus nicht.

Über das "Laufwerksbuchstabe" Auswahlfeld können Sie festlegen, welchen Laufwerksbuchstaben der jeweils einzubindende physkalische Ordner bzw. das jeweils einzubindende Image in DOSBox erhalten soll. Beim Einbinden von Disketten- und Festplattenimages können Sie außer den üblichen Laufwerksbuchstaben auch optional ein Zahl zwischen 0 und 3 angeben. Wenn Sie eine dieser Zahlen als Laufwerksbuchstabe wählen, wird das Laufwerk nicht mit einem DOSBox-Laufwerksbuchstaben versehen, sondern direkt an den virtuellen Disketten- und Festplattencontroler in DOSBox angebunden. Dies hat zur Folge, dass das Laufwerk zwar nicht direkt auf der DOSBox-Kommandozeile verwedet werden kann, aber nach dem Booten eines alternativen Betriebssystems in DOSBox als Laufwerk zur Verfügung steht.
