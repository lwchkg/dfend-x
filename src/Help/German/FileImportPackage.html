Paketmanager

Der Paketmanager ermöglicht es Ihnen, Datenpakete (Spiele, Icons für die Spieleliste, D-Fend Reloaded Sprachdateien, Iconsets für die D-Fend Reloaded Symbolleisten usw.) direkt von der D-Fend Reloaded Homepage oder von anderen Quellen herunter zu laden und zu installieren. In dem Programmfenster werden die verfügbaren Datendateien nach Typen sortiert angezeigt. Über die Installations-Schaltfläche am unteren Rand des Fensters können die ausgewählten Dateien aus dem Netz heruntergeladen und automatisch installiert werden.

Listen aktualisieren

Wenn Sie auf die "Listen aktualisieren"-Schaltfläche klicken, überprüft D-Fend Reloaded, ob auf dem Server neuere als die lokal gespeicherten Paketlisten vorliegen und läd ggf. die neueren Paketlisten herunter. Wenn Sie beim Klicken auf "Listen aktualisieren" die Umschalttaste gedrückt halten, aktualisiert D-Fend Reloaded die lokalen Paketlisten vom Server, auch wenn das Datumsfeld der Listen auf dem Server angibt, dass diese nicht geändert wurden.

Quelleneditor

In dem über die Schaltfläche "Quellen" erreichbaren Quelleneditor können Sie neben der offiziellen D-Fend Reloaded Paketeliste weiter Quellen für Datenpakete hinzufügen. Wenn Sie z.B. über die Funktion Datenpakete-Listendatei erstellen eine eigene Sammlung von Spielepaketen erstellt haben und die Listendatei zusammen mit den Datenpaketen auf einer CD archiviert haben, so können Sie hier den Pfad zu dieser Listendatei angeben. (D-Fend Reloaded sucht unabhängig vom angegebenen Luafwerksbuchstaben bei der Installation automatisch in allen verfügbaren CD-Laufwerken nach der angegebenen Datei.) Sie können hier auch Quellen im Internet angeben; beachten Sie dabei aber, dass dies neben rechtlichen vor allem auch Sicherheitsprobleme mit sich bringt. Wenn Sie Datenpakete, die Programmcode enthalten, aus dem Netz aus einer nicht vertrauenswürdigen Quelle herunterladen, gefährden Sie damit Ihr System!
