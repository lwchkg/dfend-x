Profil kopieren

Anstatt für jedes Spiel ein neues Profil anzulegen, können Sie auch ein bestehendes Profil kopieren. Beim Kopieren des Profil wird nur der Eintrag in der Spieleliste kopiert, die sonstigen Daten (insbesondere die Spieledateien) werden nicht dubliziert. In dem "Profil kopieren"-Dialog können Sie neben dem Namen für das neue Profil zwei optionale Einstellungen vornehmen:

  • Neuen Screenshot-Ordner festlegen
    Wenn diese Option markiert ist, wird basierend auf dem Namen des neuen Profils für das neue Profil automatisch ein Ordner festgelegt, in dem Screenshots, Sounds und Videos abgelegt werden können. Andernfalls verweist der Screenshot-Ordner-Eintrag des neuen Spiels auf den selben Ordner, wie das bisherige Profil (d.h. in diesem Fall wird ein Ordner von beiden Profilen gemeinsam genutzt).
  • Datenordner-Eintrag entfernen
    Wenn diese Option markiert ist, wird bei dem neuen Profil der Datenordner-Pfad entfernt. Ist diese Option deaktiviert (und besitzt das Ausgangsprofil einen Datenordner), so verweisen in Zukunft beide Profile auf den selben Ordner.
