Imagedatei erstellen

In diesem Dialog können Sie neue, leere Disketten- und Festplattenimage-Dateien erstellen. Diese Imagedateien können dann über die Laufwerke-Dialogseite des Profileditor-Dialogs als Laufwerke in DOSBox eingebunden werden.

Diskettenimages / Festplattenimages

Über die beiden Registerreiter am oberen Fensterrand können Sie auswählen, ob ein Disketten- oder ein Festplattenimage erstellt werden soll. Bei Diskettenimages können Sie den Diskettentyp oder manuell die Anzahl an Spuren, Köpfen und Sektoren pro Spur angeben. Bei Festplattenimages kann die Größe im MB angegeben werden.

Optionen

  • Imagedatei im NTFS-Dateisystem komprimieren
    Wenn diese Option aktiviert ist, wird das Image als komprimierte Datei gespeichert. Dies bedeutet, dass die Datei nur soviel Platz auf dem Datenträger belegt, wie wirklich zum Speichern der in dem Image abgelegten Dateien notwenidg ist. Die leeren Sektoren innerhalb des Images belegen fast keinen Platz. Die Kompression erfolgt durch den Windows-Festplattentreiber, d.h. Sie können mit der komprimierten Datei normal arbeiten, ohne ein (De-)Kompressionsprogramm verweden zu müssen.
  • Image bootfähig machen
    Wenn diese Option aktiviert ist, überträgt D-Fend Reloaded die FreeDOS Boot-Dateien auf das Image und erstellt einen Bootsektor auf dem Image, welches es bootfähig macht. (Um diese Option verwenden zu können, muss FreeDOS installiert und der Pfad zu FreeDOS auf der Dialogseite FreeDOS des the Programmeinstellungen-Dialogs eingetragen sein.)
  • Die DOSBox-Tastaturbelegung verwenden
    Wenn diese Option aktiviert ist, überträgt D-Fend Reloaded den FreeDOS-Tastaturtreiber in das Image und fügt eine Zeile in die autoexec.bat-Datei ein, um das Tastaturlayout, welche in der in D-Fend Reloaded gewählten Sprachdatei definiert ist, zu aktivieren.
  • Maustreiber einbinden
    Wenn diese Option aktiviert ist, überträgt D-Fend Reloaded den FreeDOS-Maustreiber in das Image und fügt eine Zeile in die autoexec.bat-Datei ein, um ihn beim Starten des Images zu aktivieren.
  • Speichermanager verwenden
    Ist diese Option aktiviert, so werden die Speicherverwaltungsprogramme himem und emm386 auf das Image kopiert und aktiviert, so dass mehr konventioneller Speicher zur Ausführung von Anwendungen zur Verfügung steht.
  • Datenträger-Verwaltungsprogramme hinzufügen
    Wenn diese Option aktiviert ist, werden Programme zum partitionieren, formatieren und prüfen von Datenträgern in das Image kopiert.
  • Texteditor hinzufügen
    Wenn diese Option aktiviert ist, wird ein Texteditor mit in das Image kopiert.
  • DosZip Dateimanager hinzufügen
    Wenn diese Option aktiviert ist, wird der DosZip Commander, ein Dateimanager, mit in das Image kopiert.
  • DOSBox-Profil zum Starten der Imagedatei anlegen
    Wenn diese Option aktiviert ist, legt D-Fend Reloaded direkt nach der Erstellung der Imagedatei ein Profil, über das die Imagedatei direkt gestartet werden kann, an.
