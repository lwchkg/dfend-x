Programmeinstellungen-Dialogseite "Programmoberfläche"

Auf dieser Seite gibt es lediglich eine Option (alle weiteren Einstellungen zum Erscheinungsbild des Programms werden über die Unterkategorien dieser Seite eingestellt): There is only one option to change on this page:

  • Fenstergröße beim Programmstart
    In diesem Feld können Sie das Aussehen des Programmfensters beim Programmstart von D-Fend Reloaded festlegen. Sie können zwischen "Vorgabewert für Fenstergröße", "Letzte Fenstergröße wiederherstellen", "Minimiert starten", "Maximiert starten" und "Vollbildmodus ohne Fensterrahmen" wählen.
    (Vorgabewert für diese Option: "Vorgabewert für Fenstergröße")
  • Icon-Zusammenstellung
    In diesem Auswahlfeld können Sie festlegen, wie die in der Symbolleiste, im Menü und in den Dialogen anzuzeigenden Icons aussehen sollen. Die gewählte Icon-Zusammenstellung wird jeweils beim Schließen des Programmeinstellungen-Dialogs aktiviert.
    (Vorgabewert für diese Option: "Modern icon set")

Die mit (*) gekennzeichnete Option ist nur sichtbar, wenn der Konfigurationsmodus "Vollständig" gewählt ist.
