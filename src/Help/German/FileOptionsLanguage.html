Programmeinstellungen-Dialogseite "Sprache"

Auf dieser Seite stehen drei Auswahlfelder zur Konfiguration der Programmsprache zur Verfügung:

  • Programmsprache
    In diesem Auswahlfeld können Sie die Sprache auswählen, in der die D-Fend Reloaded Programmoberfläche (Menüs, Dialogboxen usw.) angezeigt werden soll. Änderungen in diesem Auswahlfeld führen zu einer sofortigen Umstellung der Programmsprache, ohne dass dafür der Dialog geschlossen werden müsste oder gar das Programm neu gestartet werden müsste.

    Um eine neue Sprachdatei für die Verwendung zur Darstellung der Programmoberfläche anzulegen, müssen Sie nur auf die "Neue Sprache anlegen"-Schaltfläche klicken und in dem Übersetzungseditor Namen für die angezeigten Bezeichner in Ihrer Sprache eingeben. Zur Erstellung einer neuen Sprache für D-Fend Reloaded brauchen Sie keinerlei externe Programme (auch keinen Texteditor).

    Um eine neue (z.B. aus dem Internet heruntergeladene) Sprachdatei zu Ihrer D-Fend Reloaded Installation hinzuzufügen, müssen Sie diese lediglich in das "lang"-Unterverzeichnis des D-Fend Reloaded Datenordners (üblicherweise "C:\Users\IhrName\D-Fend Reloaded) kopieren und den Programmeinstellungen-Dialog neu öffnen. Nun sollten Sie in der Lage sein, die neu hinzugefügte Sprachdatei in dem Auswahlfeld auszuwählen.
  • DOSBox-Sprachdatei
    In diesem Auswahlfeld können Sie die Sprache, die DOSBox verwenden soll, um Nachrichten anzuzeigen, auswählen. D-Fend Reloaded zeigt in diesem Auswahlfeld alle verfügbaren *.lng Dateien aus dem DOSBox-Verzeichnis an. Um eine neue (z.B. aus dem Internet heruntergeladene) DOSBox-Sprachdatei zu dieser Auswahlliste hinzuzufügen, müssen Sie nur die *.lng in Ihr DOSBox-Programmverzeichnis kopieren und dann den Programmeinstellungen-Dialog neu öffnen.

    Die auf dieser Dialogseite ausgewählte Sprache wird bei Verwendung der Standard-DOSBox-Installation verwendet (siehe DOSBox-Seite des Programmeinstellungen-Dialogs).
  • Installersprache
    In diesem Auswahlfeld können Sie die Sprache, die in Installerpaketen verwendet werden soll, auswählen. Um Änderungen an dieser Einstellung vornehmen zu können, muss schreibend auf den für alle Benutzer des Computers gültigen Teil der Registry zugegriffen werden. D.h. für Änderungen dieser Option werden Admin-Rechte benötigt. Unter Windows Vista erscheint daher ein Bestätigungsdialog, so bald Sie den Wert in diesem Auswahlfeld ändern.
