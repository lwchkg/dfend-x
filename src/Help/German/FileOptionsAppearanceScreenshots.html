Programmeinstellungen-Dialogseite "Medienvorschau"

Diese Dialogseite ist nur sichtbar, wenn Sie den Konfigurationsmodus "Vollständig" unten im Programmeinstellungen-Dialog gewählt haben.

Auf dieser Seite können Sie das Aussehen des Screenshots-, Sounds- und Videos-Vorschaubereiches unten im Hauptfenster anpassen.

  • Hintergrundfarbe / Hintergrundbild
    Über die drei Optionen "Standard-Hintergrundfarbe", "Hintergrundfarbe" und "Hintergrundbild" kann eingestellt werden, ob D-Fend Reloaded die von Windows vorgegebene Standardhintergrundfarbe für den Vorschaubereich verwenden soll, die hier angegebene benutzerdefinierte Farbe verwenden soll oder eine Bilddatei als Hintergrund anzeigen soll.
    (Vorgabewert für diese Option: "Standard-Hintergrundfarbe") Via the three radio buttons "Default background color", "Background color" and "Background image" you can set up whether D-Fend Reloaded should you the default Windows background color in the screenshot, sounds, videos and data preview area, a custom color or a custom image.
    (Default value for this option: "Default background color")
  • Schriftgröße und Schriftart
    In diesem Feld können Sie die Schriftgröße, die für die Beschriftungen der Vorschaubilder usw. verwendet werden sollen, einstellen.
    (Vorgabewert für diese Option: 9)
  • Screenshots-Vorschaugröße
    Die Screenshots werden auf die in diesem Feld angegebene Höhe und Breite skaliert. Das Seitenverhältnis bleibt dabei erhalten.
    (Vorgabewert für diese Option: 100)
