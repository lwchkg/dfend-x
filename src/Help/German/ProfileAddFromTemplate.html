Hinzufügen von Vorlage

Wenn Sie diesen Menüpunkt wählen, öffnet sich ein weiteres Untermenü, welches alle im Vorlageneditor konfigurierten Vorlegen enthält. Durch das Anklicken einer der Vorlagen aus diesem Untermenü können Sie ein neues DOSBox-Profil anlegen, welches dann auch sofort im Profileditor-Dialog geöffent wird.
