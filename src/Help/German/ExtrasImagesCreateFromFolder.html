Imagedatei von Ordnerinhalt erstellen

In diesem Dialog können Sie Diskettenimages mit dem Inhalt eines Ordners auf der Festplatte erstellen.

  • Imagetype
    In dieser Box können Sie auswählen, ob ein Disketten- oder ein Festplattenimage angelegt werden soll. Diskettenimages sind 1,44MB groß, Festplattenimages werden automatisch an die Größe der zu kopierenden Dateien angepasst (wobei zusätzlich 5MB freier Platz einkalkuliert werden).
  • Image bootfähig machen
    Wenn diese Option aktiviert ist, überträgt D-Fend Reloaded die FreeDOS Boot-Dateien auf das Image und erstellt einen Bootsektor auf dem Image, welches es bootfähig macht. (Um diese Option verwenden zu können, muss FreeDOS installiert und der Pfad zu FreeDOS auf der Dialogseite FreeDOS des the Programmeinstellungen-Dialogs eingetragen sein.)
  • Die DOSBox-Tastaturbelegung verwenden
    Wenn diese Option aktiviert ist, überträgt D-Fend Reloaded den FreeDOS-Tastaturtreiber in das Image und fügt eine Zeile in die autoexec.bat-Datei ein, um das Tastaturlayout, welche in der in D-Fend Reloaded gewählten Sprachdatei definiert ist, zu aktivieren.
  • Maustreiber einbinden
    Wenn diese Option aktiviert ist, überträgt D-Fend Reloaded den FreeDOS-Maustreiber in das Image und fügt eine Zeile in die autoexec.bat-Datei ein, um ihn beim Starten des Images zu aktivieren.
  • Speichermanager verwenden
    Ist diese Option aktiviert, so werden die Speicherverwaltungsprogramme himem und emm386 auf das Image kopiert und aktiviert, so dass mehr konventioneller Speicher zur Ausführung von Anwendungen zur Verfügung steht.
  • Datenträger-Verwaltungsprogramme hinzufügen
    Wenn diese Option aktiviert ist, werden Programme zum partitionieren, formatieren und prüfen von Datenträgern in das Image kopiert.
  • Texteditor hinzufügen
    Wenn diese Option aktiviert ist, wird ein Texteditor mit in das Image kopiert.
  • Fertiges Image auf reale Diskette schreiben
    Wenn diese Option aktiviert ist, wird das Diskettenimage nach seiner Erstellung auf eine reale Diskette geschrieben.
