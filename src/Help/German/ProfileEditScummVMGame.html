Profileditor-Dialogseite "Spielespezifische Einstellungen"

Auf diese Dialogseite können Sie verschiedene Einstellungen vornehmen, die jeweils nur für einige Spiele zur Verfügung stehen. Die Namen der Spiele, für die die Einstellungen jeweils gültig sind, werden jeweils unterhalb der Auswahlfelder angezeigt. Außerdem werden die Auswahlfelder, die für das jeweils gewählte Spiel nicht unterstützt werden, deaktiviert.
