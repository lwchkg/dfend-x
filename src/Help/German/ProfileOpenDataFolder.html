Datenordner öffnen

Wenn Sie diesen Menüpunkt anklicken, wird das Datenverzeichnis des gewählten Spiels im Windows Explorer geöffnet. Das Datenverzeichnis eines Profils kann auf der Verzeichnisse-Dialogseite des Profileditor-Dialogs festgelegt werden. In dem Datenverzeichnis können Sie zusätzliche, zu dem Spiel gehörende Dateien wie z.B. Handbuchdateien ablegen.

Über den Menüpunkt Datei in Datenordner öffnen im "Profil"-Menü können Sie auch direkt einzelne Dateien aus dem Datenordner öffnen.
