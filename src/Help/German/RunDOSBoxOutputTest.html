DOSBox Ausgabemethoden testen

In diesem Fenster können Sie die zur Verfügung stehenden DOSBox Ausgabemethoden testen. Nicht jeder Ausgabemethode wird von jedem Grafiktreiber unterstützt. Ist in einem Profil eine nicht unterstützte Ausgabemethode gewählt, so startet das entsprechende Profil nicht.

Die jeweils zu verwendende Ausgabemethode kann auf der Profileditor-Dialogseite Grafikkarte in dem Feld "Darstellungsmethode" eingestellt werden. Alternativ kann der Dialog Mehrere Profile gleichzeitig bearbeiten verwendet werden, um mehrere oder alle Profile auf einmal zu konfigurieren.
