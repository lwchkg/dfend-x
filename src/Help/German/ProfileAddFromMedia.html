Spiel von Quellmedien installieren

Über diese Option können Sie Spiele zur D-Fend Reloaded Spieleliste hinzufügen, die zunächst noch installiert werden müssen. In dem Dialog können Sie angeben, von welchem Typ das Installationsmedium bzw. die Installationsmedien sind und können die Pfade zu diesem bzw. diesen Medien angeben. Wenn Sie auf "Ok" klicken, wird D-Fend Reloaded die entsprechenden Medien temporär einbinden und DOSBox im Fenstermodus starten. In DOSBox wird, sofern D-Fend Reloaded ein eindeutiges Installationsprogramm auf dem ersten Installationsmedium findet, das Installationprogramm bereits gestartet. Sie müssen mit Hilfe dieses normalen Installationsprogrammes das Spiel nun lediglich in einem beliebigne Verzeichnis auf dem Laufwerk C: installieren. Nach dem Ende der Installation überprüft D-Fend Reloaded das Spieleverzeichnis automatisch auf neue Unterordner und fügt das neue Spiel zur Spieleliste hinzu.
