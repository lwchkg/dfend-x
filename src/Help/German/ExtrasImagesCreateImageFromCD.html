ISO-Image von CD erstellen

In diesem Dialog können Sie ISO-Imagedateien von bestehenden CD-ROMs erstellen. ISO-Images können uuml;ber die Dialogseite Laufwerke des Profileditor-Dialogs als Laufwerke in DOSBox eingebunden werden.

Der D-Fend Reloaded ISO-Image-Leser kann nur den jeweils ersten Track einer CD lesen. Wenn Sie ein ISO-Image einer CD, die mehrer Tracks besitzt, erstellen wollen, sollten Sie einen externen ISO-Image-Leser verwenden.
