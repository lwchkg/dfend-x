D-FendReloaded Portable Help

Running DOS games has never been easier...

D-FendReloaded Portable is D-Fend Reloaded is a DOSBox frontend for the Windows platform, D-FendReloaded packaged with a PortableApps.com launcher as a portable app, so you can install it on your iPod, USB flash drive, portable hard drive, etc. and play old games on any Windows PC you can connect to. It leaves no personal information behind on the machine you run it on, so you can take it with you wherever you go. Even so DOSBox is easy to handle compared to most of the other emulators, there is no graphical environment to setup the hardware used by the games and if you want to run different games needing different hardware setups you need to keep different versions of the DOSBox configuration file and always need to start DOSBox using the right configuration file. Long time ago there has been a freeware frontend called "D-Fend". But some day the development of this program has been stopped and it has been removed from the internet. D-Fend Reloaded is a complete rewrite of D-Fend. It is nearly 100% compatible with D-Fend (you can upate from D-Fend to D-Fend Reloaded by just replacing the program file if you want to). But by now D-Fend Reloaded has many new features D-Fend never had. D-FendReloaded section of the Authors' WebSite. Learn more about D-FendReloaded...
