D-FendReloaded Portable Aide

L'exécution des Jeux DOS n'a jamais été aussi facile...

D-FendReloaded Portable est D-Fend Reloaded est une interface de DOSBox pour la plate-forme Windows, D-FendReloaded progiciel avec un lanceur PortableApps.com comme une application portable app, vous pouvez donc l'installer sur votre iPod, clé USB, disque dur portable, etc. et jouer à des jeux anciens sur n'importe quel PC Windows, que vous pouvez vous connecter. Il ne laisse aucune information personnelle derrière sur la machine que vous l'exécutez, vous pouvez donc l'emporter avec vous partout où vous allez. Même si DOSBox est facile à utiliser par rapport à la plupart des autres émulateurs, il n'y a aucun environnement graphique pour configurer le matériel utilisé par les jeux et si vous voulez faire tourner des jeux nécessitant différentes configurations matérielles différentes, vous devez garder des versions différentes de la configuration de DOSBox fichier et toujours besoin de commencer à DOSBox en utilisant le bon fichier de configuration. Il y a longtemps il y a eu une interface gratuite appelé "D-Fend". Mais un jour le développement de ce programme a été arrêté et il a été retiré de l'internet. D-Fend Reloaded est une réécriture complète de D-Fend. Il est presque 100% compatible avec D-Fend (vous pouvez Mettre À Jour de D-Fend à D-Fend Reloaded simplement en remplaçant le fichier du programme si vous voulez). Mais maintenant D-Fend Reloaded a de nombreuses nouvelles fonctionnalités D-Fend jamais eu. D-FendReloaded section du site Web de l'auteur. En savoir plus sur D-FendReloaded...
